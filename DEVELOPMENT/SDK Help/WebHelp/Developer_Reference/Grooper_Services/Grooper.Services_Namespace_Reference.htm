Grooper.Services Namespace Reference

Classes

Class Name Description
ImportWatcher Import Watcher is a service which polls an Import Provider for new documents, and imports them based on rules.
LicenseRenewalService Performs usage reporting when the Grooper pay-as-you-go licensing model is in use.
LicenseService Distributes runtime licenses to Grooper applications.
SystemMaintenanceService Provides Grooper system maintenance services.
TaskProcessingService The Task Processing service executes Unattended Activities, which perform the work associated with a step in a Batch Process.
TimerService Provides a base class for services which perform work on a timer interval.
WebService Provides a base class for services which act as self-hosted web services.
