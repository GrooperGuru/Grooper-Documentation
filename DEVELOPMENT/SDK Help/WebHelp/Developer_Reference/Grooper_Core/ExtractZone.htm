GetRuntimePageNo(NumPages As Int32) As Int32 Generates the runtime page number for a document with N pages. The Page No value of a CaptureZone can be positive or negative. Positive numbers represent an actual page number, while negative numbers represent "reverse" page numbers. For example, a Page No of -1 represents the last page in the document, -2 represents the 2nd to last page, and so on. This method calculates and returns the actual page number.
