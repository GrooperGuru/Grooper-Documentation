Grooper.CMIS Namespace Reference

Classes

Class Name Description
CmisBatchTransfer Transfers a Batch to another Grooper repository, optionally launching the batch into a Batch Process in the target repository.
CmisConnection A CMIS Connection object represents a connection to CMIS-compliant content server. For a detailed overview of CMIS, please refer to the Content Management System Connectivity topic.
CmisContentType A CMIS Content Type represents a content type defined in a CMIS Repository. These objects are not created directly, but created when content types are imported from the property panel of a CMIS Repository object. CMIS Content Types are used to create and maintain mappings between Grooper content types and content types in a repository.
CmisDocument
CmisExport Provides export connectivity to CMIS-compliant content management platforms.
CmisFolder
CmisImport Provides import connectivity to CMIS-compliant content management platforms.
CmisLink Defines a reference to a document in a CMIS Repository.
CmisObject
CmisPropertyBooleanDefinition Represents a boolean (true / false) property value.
CmisPropertyDateTimeDefinition Represents a date / time property value.
CmisPropertyDecimalDefinition Represents a decimal property value.
CmisPropertyDefinition The base class for all CMIS property definitions.
CmisPropertyHtmlDefinition Represents a property capable of storing HTML.
CmisPropertyIdDefinition Represents a unique identifier property value.
CmisPropertyIntegerDefinition Represents an integer property value.
CmisPropertyStringDefinition Represents a string property value.
CmisPropertyUriDefinition Represents a URI property value.
CmisRepository A CMIS Repository object represents a logical repository managed by a CMIS server. CMIS Repository objects are not created directly - they are created by importing a repository using the property panel of a CMIS Connection object. For a detailed overview of CMIS, please refer to the Content Management System Connectivity topic.
CmisTypeDescriptor Used to dynamically enable/disable or show/hide object properties in a PropertyGrid control.
