Grooper Namespace Reference

Classes

Class Name Description
BorderExF Represents the size of a border on each edge of a rectangle.
ConnectedObject Represents the base class for objects stored as serialized properties which require an IConnected implementation.
DataSource Repository connections define the parameters for connecting to a Grooper database.
EmbeddedObject Represents an Object that is embedded inside of another object.
FileStore A File Store defines a storage location where Grooper can store content files associated with nodes in the Grooper tree structure.
Folder Represents a folder object in which objects can be stored.
GrooperNode Grooper Nodes are objects which appear in the Grooper Tree Structure. Every object at every level of the tree structure, including folders, are considered Grooper Nodes.
GrooperRoot Represents the root node of a Grooper repository.
GrooperRoot_Stats: SessionQueryCriteria Defines search criteria for session statistics.
InfrastructureFolder Represents the 'Infrastructure' folder found in the root of a Grooper repository.
LicenseServer A License Server object is used to store connection information for a running instance of Grooper License Server.
Machine The Machine object represents the a server or workstation in the Grooper Enterprise. Selecting a Machine in the Machines Folder will display a panel with a performance section, a Services Installed section and a Details section.
MachinesFolder The Machines Folder provides a consolidated list of servers and workstation in the infrastructure, as well as a consolidated list of service instances installed. From this panel, critical statistics such as processor usage, disk space usage, and memory usage can be viewed for any machine, and services on any machine (or all machines) can be started and stopped.
ObjectCommand Object Commands perform specialized tasks on one or more Grooper Objects.
ObjectLibrary An Object Library is a .Net library of custom objects to be used in Grooper.
RectangleExF Represents a rectangle using double-precision floating point coordinates.
ServiceInstance Service Instances represent processes which may be installed as Grooper services, such as the Grooper Licensing service or the Task Processing service.
ShellExecuteInfo Represents information obtained from the Windows Registry regarding a shell verb.
SizeExF Represents height and width as double values.
ViewableDatabaseObject The base type for all objects that reside in the Grooper Database.
