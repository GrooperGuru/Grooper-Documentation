Grooper.Messaging Namespace Reference

Classes

Class Name Description
FtpExport Provides export connectivity over the Ftp and Ftps protocols.
FtpImport Provides import connectivity to ftp servers.
FtpLink Defines a reference to a file on an Ftp file server.
MailExport Provides export connectivity to IMAP mail servers.
MailImport Provides mail message import capabilites over the IMAP protocol.
MailLink Defines a reference to an email on an IMAP mail server.
MailMimeTypeHandler Handles the message/rfc822 Mime Type.
SftpExport Provides export connectivity over the SFTP protocol.
SftpImport Provides import connectivity to SSH/SFTP servers.
SftpLink Defines a reference to a file on a SSH/SFTP file server.
