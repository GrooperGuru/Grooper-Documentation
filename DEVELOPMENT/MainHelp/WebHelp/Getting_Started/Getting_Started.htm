Grooper 2.70 Documentation

Getting Started

Attending Grooper training is the best way to get started as a Grooper administrator.  This guide is not intended as a replacement for formal training, but rather a refresher for administrators who have completed training.  To enroll students in Grooper training classes, contact your Grooper sales representative.

This section covers the following topics:

  • Installing Grooper - How to install Grooper and initialize a Grooper repository.
  • Grooper Concepts - An overview of key concepts on which Grooper is designed.
  • How To Guide - Guides which detail how to perform common administrative tasks in Grooper.
