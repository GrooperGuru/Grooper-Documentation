Undo Image Cleanup

Every Batch Page contains at a minimum a primary image and a thumbnail image. However, Grooper allows batch pages and documents to have multiple files attached. One additional file attachment is called the Undo Image Image file attached to a Batch Page. Typically saved at scan time or during Image Processing prior to any image cleanup being performed.. An undo image is a copy of the original image file without any image cleanup having been applied. To enable the saving of an undo image at scan time, or during Image Processing, enable the option in the Scan Settings or Image Processing Settings respectively.

If an undo image exists for a batch page, then all image cleanup done to a batch page can be 'undone' using the undo command in the Page Viewer. Additionally, if a page has been modified by a user using tools in the page viewer, those changes can also be 'undone'.

If the saving of undo images is not enabled in either the scan or image processing settings, and a batch page is modified by a user using the page viewer, then the original image will be automatically saved as an undo image.
