HOW TO: Apply a Modified Batch Process to Existing Batches

If changes made to a Batch Process need to be applied to existing Batches, use the following procedure.  

Step 1 - Publish the Batch Process

  • In Grooper Administration, navigate to the Batch Process in the 'Batch Processing • Processes • Working' folder.
  • Use the 'Publish' command from the Batch Process tab to publish the Batch Process.

Step 2 - Pause the batch

  • Navigate to the batch in Grooper Administration or Grooper Dashboard.
  • In Grooper Administration, on the Batch tab of the property panel, use the Pause command from the toolbar to pause the batch. In Grooper Dashboard, use the Pause command on the Batch Info tab's toolbar to pause the batch.

Step 3 - Update the Batch Process

  • Click the Update Process command.  This will display the Select Processing Step dialog box shown below.
  • Select the step of the Batch Process where the batch should resume processing and click OK.

 

Step 4 - Resume the batch

  • Use the Resume command from the toolbar to resume processing on the batch.
