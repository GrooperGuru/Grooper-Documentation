HOW TO: Build a Batch Process

This topic provides a step-by-step guide to creating a new Batch Process.

Step 1: Create a New Batch Process

  • In Grooper Administration, navigate to 'Batch Processing • Processes • Working'.
  • Right-click on the folder and select 'Add Batch Process...'.
  • Enter a unique name for the Batch Process and click OK.

 

Step 2: Add Processing Steps

  • Use the "Add Step" command to add processing steps to the Batch Process.
  • When all desired steps have been added, click the Save button.

 

Step 3: Publish

  • Use the Publish command to publish the Batch Process.
  • The Batch Process should now be available when creating new batches.
