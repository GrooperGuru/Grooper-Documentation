Install an SSL Certificate

To enable HTTPS for a Grooper Web Service (such as Grooper CMIS Server) an X.509 certificate must be installed on the host machine and bound to a port number.
 
Complete the following steps to bind a port number to an installed X.509 certificate:

1)     View the certificate with the MMC Snap-in

2)     Get the certificate's thumbprint

3)     Build the command line command and parameters

a)     Command line format:  netsh http add sslcert ipport=0.0.0.0:<Port Number> certhash=<Thumbprint> appid={00112233-4455-6677-8899-AABBCCDDEEFF}

i)        ipport=0.0.0.0 - It is unnecessary to indicate an actual IP address here because our only intent is to associate the Port Number to the installed certificate. So, leave this as 0.0.0.0

ii)      <Port Number> - The port number for the Grooper service

iii)     certhash - The certhash is the thumbprint of the certificate without any spaces

iv)    appid - This app id is irrelevant. so. leave this as {00112233-4455-6677-8899-AABBCCDDEEFF}

4)     Execute the command in a command prompt window

View the certificate with the MMC Snap-in:
  1. Open a command Prompt window
  2. Type mmc and press the ENTER key
  3. On the File menu, click Add/Remove Snap In
  4. Click Add
  5. In the Add Standalone Snap-in dialog box, select Certificates
  6. Click Add
  7. In the Certificates snap-in dialog box, select Computer account and click Next
  8. In the Select Computer dialog box, click Finish
  9. In the Add or Remove Snap-ins dialog box, click OK
  10. On the Add/Remove Snap-in dialog box, click OK
  11. In the Console Root window, click Certificates (Local Computer) to view the certificate stores for the computer

Get the certificate's thumbprint:

  1. Double-click the certificate in the MMC Snap-in console
  2. Click the Details tab
  3. Scroll to the bottom of the Field Value list and click Thumbprint
  4. Copy the thumbprint of the certificate into a text editor such as Notepad
  5. Remove all spaces between the hexadecimal characters
