HOW TO: Perform Real-Time Image Cleanup

This topic discusses how to perform real-time image cleanup during scanning.

Step 1 - Define or select an IP Profile

  • In Grooper Administration, navigate to 'Profiles â€¢ IP Profiles'.
  • Check to see if an IP Profile already exists with the desired settings.  If so, skip to Step 2.
  • Create a new IP Profile and add the desired image processing commands.
  • Save the IP Profile.

Step 2 - Assign the IP Profile to the scan module.

  • Navigate to the Batch Process to be used in Grooper Administration.
  • Highlight the Scan step, and click the 'Edit Step...' button.
  • Click the 'Module Settings...' button in the Batch Process Step dialog box.
  • Select the IP Profile created or selected in Step 1 in the 'Image Processing (IP) Profile' drop-down.
  • Click OK to close all dialog boxes.
  • Save and publish the Batch Process using the commands on the toolbar.
