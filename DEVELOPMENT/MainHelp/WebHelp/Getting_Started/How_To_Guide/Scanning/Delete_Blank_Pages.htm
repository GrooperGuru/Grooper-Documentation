HOW TO: Delete Blank Pages

This topic describes how to configure the Grooper Scan Module to delete blank pages during scanning.

Step 1 - Define or select a Separation Profile

  • In Grooper Administration, navigate to 'Profiles â€¢ Separation Profiles'.
  • Check to see if a Separation Profile already exists with the settings shown below.  If so, skip to Step 2.
  • Create a new Separation Profile.
  • Add an event to delete blank pages.  Click the Add Event toolbar command, and set the event properties as shown below, then click OK.

 

 

Step 2 - Assign the Separation Profile to the Scan Module

  • Navigate to the Batch Process to be used in Grooper Administration.
  • Highlight the Scan step, and click the 'Edit Step...' button.
  • Click the 'Module Settings...' button in the Batch Process Step dialog box.
  • Select the Separation Profile created or selected in Step 1 in the 'Separation Profile' drop-down.
  • Click OK to close all dialog boxes.
  • Save and publish the Batch Process using the commands on the toolbar.
