HOW TO: Enable ScanOnce™

This topic describes how to enable ScanOnce™ during scanning.

Step 1 - Define or select a Scanner Profile

  • In Grooper Administration select the Scanner Profiles Folder or open the Scan Module.
  • Select or Define a Scanner Profile:

    • Select a Scanner Profile

      • Click the Scanner_Profile_Selection dropdown menu and select a Scanner Profile.

 

 

    • Define a Scanner Profile

      • Click the Scanner Profile Configuration dropdown menu and click Add...

 

 

      • Select ISIS or TWAIN from the device type dropdown list.
      • Select the desired scanner model from the Select Device list.
      • Provide a name for the Scanner Profile in the Profile Name text box.
      • Click Create.

 

 

      • Provide a name for the new scanner profile and click OK.
  • Click the Scanner Profile Configuration dropdown menu and click Settings...
  • Click the ScanOnce_Settings tab.

 

 

  • The options for ScanOnce configurations are explained in detail in the ScanOnce Settings portion of the Scan Control topic.
  • Click OK.
