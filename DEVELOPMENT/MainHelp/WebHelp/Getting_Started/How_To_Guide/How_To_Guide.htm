Grooper 2.70 Documentation

How To Guide

The topics below provide step-by-step instructions for performing common tasks in Grooper.

Getting Started Tutorials

  • Scan, Index, and Export - Walks through the process of building a simple 3-step Batch Process which performs scanning, manual data entry, and an XML/PDF file system export.

Configuration and Setup

  • Estimate System Sizing Metrics - Provides information to help IT planners properly plan for the number server processing cores, amount of file storage space, and amount of database storage space needed to process various volumes of documents.
  • Install a Processing Module as a Service - Install processing modules such as Image Processing, Full Text OCR, Extract, and etc. to run as Windows services.
  • Install a Stand-Alone Instance of Grooper - Install Grooper to run as a stand-alone application with its own local instance of SQL Server.
  • Install an SSL Certificate - To enable HTTPS for any Grooper service, an SSL certificate must be installed on the server hosting the service.

Batch Processing

  • Build a Batch Process - Create a new Batch Process defining steps that batches will be processed through.
  • Apply a Modified Batch Process to Existing Batches - Apply changes made to a Batch Process to existing batches.
  • Automatically Delete Completed Batches - Configure Batch Processes to automatically delete batches when they have successfully complete all steps.
  • Modify the Flow of a Batch Process - Learn how to use Grooper's scripting interface to customize the flow of batches through a Batch Process.

Scanning

  • Use Separator Sheets - Separate documents in real time during scanning using patch code or barcode separator sheets.
  • Perform Real-Time Image Cleanup - Perform real-time image cleanup on images as they are scanned.
  • Delete Blank Pages - Configure Grooper to delete blank pages.
  • Enable Undo - Enable the 'undo' feature which allows images to be reverted back to their original state as they came off the scanner, before any image processing was applied.
  • Enable ScanOnce™ - Set up the ScanOnce™ feature of Grooper, which captures a full-color version of every page in the background, allowing operators to change the color format, adjust the threshold, and adjust the brightness and contrast without rescanning the document.

Image Processing

  • Perform Real-Time Image Cleanup - Perform real-time image cleanup on images as they are scanned.

Classification & Separation

  • Configure Visual Classification - Use visual classification to classify documents by how they look.  Can be used to classify and separate structured documents in real time during scanning.

Data Extraction

  • Perform Database Lookups - Perform database lookups to validate or populate field values.
  • Validate Data Using a Script - Perform custom validation of data elements using Grooper's scripting interface.
