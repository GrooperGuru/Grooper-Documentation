Grooper 2.70 Documentation

Service Manager Control

Provides a user interface for configuring the services which run on a Grooper workstation or server.

Toolbar Commands

Command Name Description
Add... Add a new service instance.
Edit... Edit the properties of the selected service instance.
Start Start the selected service.
Start All Start all services.
Stop Stop the selected service.
Stop All Stop all services.
Tools / Add Multiple Task Processing Services... Add a multiple instances of the Task Processing Service.
  Tools / Delete Multiple Services / Delete All Services Uninstalls all service instances.
  Tools / Delete Multiple Services / Delete All Task Processing Services Uninstalls all instances of the Task Processing Service.
