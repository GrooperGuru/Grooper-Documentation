Grooper 2.70 Documentation

Data Source Manager Control

Provides a user interface for configuring database connections to Grooper Repositories.

Toolbar Commands

Command Name Description
Cancel Cancel changes.
Initialize Initialize a new Grooper repository using the selected repository connection.
Refresh Refresh the list of repository connections.
Save Save changes.
Set Default Set the selected repository as the default repository.
Test Connection Tests the selected connection. Please note that testing a connection will also update the repository name in the list of repositories. If the name of a Grooper repository is changed in Grooper Administration, the name reflected in the list of connections can only be updated by testing the connection in Grooper Configuration.
Tools / Install SQL Express Install an instance of SQL Server Express on the local machine.

Context Menu Commands

Command Name Description
Purge Deletes all database tables, and clears the file store for the selected repository, returning it to a pre-initialized state. WARNING: This command will delete all data in the selected repository!
