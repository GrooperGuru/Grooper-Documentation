Grooper Dashboard

Grooper Dashboard provides batch management and reporting capabilities for production batches. This application is used by managers, supervisors, and end users to view and manage production batches as they are processed through Grooper Batch Processes. Typical uses for Grooper Daskboard include the following:

  • Monitor system throughput and user productivity.
  • Create, edit, or delete batches.
  • Import batches of documents into Grooper using an Import Provider.
  • View reports and analytics.
  • Identify and resolve batch processing errors.
  • Start, stop, or edit Grooper services.

Child Controls

Control Type Description
Batch Manager Provides a user interface for viewing and managing production Batches.
Batch List Displays a list of batches.
Batch Info Viewer Provides a tabbed interface for viewing information about a batch.
Batch Task Status Chart Displays a bar chart showing the task processing status for each step in the Batch Process.
Batch Processing History Viewer Displays the history of processes and/or users which have performed task processing on the selected batch.
Batch Event Viewer Displays a list of events which have occurred on a Batch object.
Batch Task Viewer Displays the tasks associated with a Batch.
Batch Details Viewer Displays details about the currently selected batch.
Service Instance Viewer Displays a list of Service Instances for an individual machine, or for all machines attached to the Grooper repository.
Report Viewer Renders and displays a Report Instance.

Toolbar Commands

Command Name Description
Refresh Refeshes the currently displayed report.
