Grooper 2.70 Documentation

Classification Viewer

Displays the contents of a Batch, along with a list of Classification Candidates.

Child Controls

Control Type Description
Batch Viewer Displays the contents of a Batch in a hierarchical folder view. The Batch Viewer control is used throughout Grooper to display and edit the contents of a batch.

Context Menu Commands

Command Name Description
Assign Type Opens the Assign Document Type dialog and sets the selected [Batch Folder's] document type to the selected Content Type.

Additional Commands

  Command Name Shortcut Keys Description
Apply Image Cleanup Alt+I Applies an IP Profile to one or more pages.
Property Name Description
IP Profile The IP Profile to execute.
Assign Content Type Ctrl+F Assigns a Content Type to the currently selected folder.
Property Name Description
Content Type Content Type to assign.
Combine Loose Pages Ctrl+P Combines each range of contiguous loose pages into a new document, and attempts to classify each.
Go To Next Issue Ctrl+N Sets the selection to the next loose page or un-classified Batch Folder.
Mark As Reviewed None Marks one or more pages as reviewed. If the page was previously flagged, the flag will be removed.
Merge Pages Ctrl+M Inserts the selected pages into a new folder, optionally classified as a content type.
Property Name Description
Content Type Content Type to assign.
Reprocess Batch Ctrl+F5 Reprocess all or part of the batch through Classification.
Property Name Description
Perform Separation Reprocess loose pages using ESP™ auto separation.
Re-Classify Pages If enabled, re-runs classification on the page objects prior to performing separation. Enable this option to leverage training which did not exist at the time of classification, or to recognize newly-added document types.
Perform Classification Re-run classification. By default, this only attempts to re-classify un-classified folders.
Re-Classify All Folders Include folders which are already classified.
Revert To Loose Pages   Revert batch structure to a sequence loose pages, eliminating all foldering.
Split Document Ctrl+S Splits a folder or document at the currently selected page, optionally assigning a content type to the new document.
Property Name Description
Content Type Content Type to assign.
Train Document Ctrl+Shift+T Select a Document Type for the selected document and train the Content Model with its features.
Unclassify Ctrl+U Removes the selected folder and inserts its contents into the parent at the same position the folder was located.
Unmark Ctrl+U Marks one or more pages as reviewed. If the page was previously flagged, the flag will be removed.
