CMIS Result Set Viewer

Displays a result set from a CMIS query.

Toolbar Commands

Command Name Description
Delete All Delete all matching results.
Next Page Go to the previous result set page.
Previous Page Go to the next result set page.

Context Menu Commands

  Command Name Shortcut Keys Description
Copy Ctrl+C Copy all selected objects to the Windows clipboard.
Delete Del Delete all selected objects from the CMIS Repository.
Properties... F12 Display properties for the selected CMIS object.
Select All Ctrl+A Select all displayed results.

Additional Commands
