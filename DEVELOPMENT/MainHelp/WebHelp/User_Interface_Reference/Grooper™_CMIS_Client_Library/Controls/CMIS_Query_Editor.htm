Grooper 2.70 Documentation

CMIS Query Editor

Provides user interface for editing a CMIS query string directly. Supports syntax highlighting and code completion.

Child Controls

Control Type Description
Code Editor A text editor control supporting syntax highlighting and code completion for various languages.

Context Menu Commands

  Command Name Shortcut Keys Description
Copy Ctrl+C Copies the selected text to the Windows clipboard.
Cut Ctrl+X Cuts the selected text to the Windows clipboard.
Execute Query F5 Executes the current query.
Paste Ctrl+V Pastes text from the Windows clipboard at the current caret location.
Redo Ctrl+R Redo a text edit previously removed with the Undo command.
Undo Ctrl+U Undo the previous text edit.
