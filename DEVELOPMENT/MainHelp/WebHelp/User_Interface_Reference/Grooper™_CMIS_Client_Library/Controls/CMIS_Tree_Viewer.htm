Grooper 2.70 Documentation

CMIS Tree Viewer

Displays a branch of a CMIS Repository.

Context Menu Commands

  Command Name Shortcut Keys Description
Copy Ctrl+C Copy the selected item to the Windows clipboard.
Cut Ctrl+X Cut the selected item to the Windows clipboard.
Delete Item Del Delete the selected object from the CMIS Repository.
New Folder Ins Create new folder as a child of the selected folder.
Paste Ctrl+V Paste objects from the Windows clipboard to the currently selected CMIS folder. The following types of clipboard data may be pasted to CMIS:
  • Grooper Batch Folder and Batch Page objects
  • File system objects
Properties... Displays the properties of the selected CMIS object.
Rename Item F2 Rename the selected object.
