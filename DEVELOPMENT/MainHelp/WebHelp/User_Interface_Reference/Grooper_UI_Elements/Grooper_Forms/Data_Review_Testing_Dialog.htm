Grooper 2.70 Documentation

Data Review Testing Dialog

Provides a user interface which simulates the Data Review activity in Grooper Administration.

Child Controls

Control Type Description
Index Panel Displays Document Instance objects, providing a user interface for reviewing and editing the data elements associated with a document. The Index Panel is a hierarchy of Index Panel Control objects, designed to allow editing of the hierarchical Data Instance objects produced by the Extraction activity.
Page Viewer Displays image-based documents.
Image Viewer
Instance Viewer Displays Data Instance objects, allowing their contents to be exlored.
Property Grid
Page Viewer Displays image-based documents.
Image Viewer
Instance Text Viewer Displays the text associated with a Data Instance, optionally showing control characters.
OCR Character Viewer Displays the list of individual characters associated with a set of OCR results.

Toolbar Commands

  Command Name Shortcut Keys Description
Close Close the Data Review window.
Edit Data Model... View or edit the properties of the Data Model associated with the currently displayed document. Displays the Node Properties Dialog.
Extract Execute extraction on the currenlty displayed document.
Field / Add... Ctrl+Shift+Ins Add a new field to the Data Model. Displays the New Item Dialog to specify the name of the new field.
Field / Delete Ctrl+Shift+Del Delete the currently focused field from the Data Model.
Field / Edit... Ctrl+Shift+F12 View or edit the properties of the currenlty focused field. Displays the Node Properties Dialog.
  Field / Go To Field Object Close the Data Review screen and navigate to the currently focused field.
Field / Move Down Ctrl+Shift+Down Move the currently focused field down.
Field / Move Up Ctrl+Shift+Up Move the currently focused field up.
Move to the next document in the batch. Go to the next document.
Move to the previous document in the batch. Go to the previous document.
