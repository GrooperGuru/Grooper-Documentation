Grooper 2.70 Documentation

Grooper Forms

A list of all Grooper Forms.

Form Name Description
Add Report Instance Dialog Provides a user interface for listing Report Instance objects, allowing one to be selected.
Add Sample Image Dialog Provides a user interface for selecting sample images.
Adjust Binary Image Dialog Provides a user interface for editing the display properties of a page when the display color format is black and white.
Adjust Color Image Dialog Provides a user interface for editing the display properties of a page when the display color format is color or grayscale.
Attributes Editor Dialog Provides a user interfacing allowing creation of an attribute mask for setting or clearing attributes on nodes in the Grooper Tree Structure.
Batch Properties Dialog Provides a user interface for displaying the properties of a Batch.
Batch XML Viewer Dialog Provides a user interface for parsing and displaying the contents of a batch in hashed XML format that can be used for batch comparison. This can be useful for comparing the same batch before and after: upgrading Grooper, modifying referenced resources (data types, lexicons, etc), modifying a content model, etc.
Browse For FTP Folder Dialog Provides a user interface for browsing to a folder on an FTP Server.
Browse For Mail Folder Dialog Provides a user interface for browsing to a folder on a Mail Server.
Browse For SSH Folder Dialog Provides a user interface for browsing to a folder on an SFTP Server.
CMIS Object Properties Dialog Provides a user interface for displaying the properties of a CMIS object in a dialog box.
CMIS Query Dialog Provides a user inteface for Displaying a CMIS Repository Searcher control in a dialog box.
Context Zone Editor Dialog Provides a user interface for editing a context zone.
Context Zone List Editor Dialog Provides a user interface for editing the context zones associated with a Field Class.
Data Instance Viewer Dialog Provides a user interface displaying a Data Instance hierarchy in a dialog box.
Data Review Flag Page Dialog Provides a user interface for specifying a reason why a page was flagged in the Grooper Data Review Module.
Data Review Testing Dialog Provides a user interface which simulates the Data Review activity in Grooper Administration.
Edit CMIS Connection Dialog Provides a user interface for editing and testing an AX Web Services Connection.
Extract Text Dialog Provides a user interface for selecting a range of text content from a document.
Extraction Test Results Dialog Provides a user interface for Displaying the results of an extraction test.
Field Profile Test Dialog Provides a user interface allowing a Field Profile to be tested against test batches.
Field Zone Test Dialog Provides a user interface for testing Field Zones.
File Name Editor Dialog Provides a user interface displaying an open file dialog and allows the editing of a file path.
Find Objects Dialog Provides a user interface for searching the Grooper Tree Structure by name, type, or ID.
Folder Path Editor Dialog Provides a user interface displaying a folder browser dialog and allows the editing of a folder path.
frm Export Node List Editor
frm Published Nodes Manager
frm Set Password
Fuzzy Match Visualizer Dialog Provides a user interface for searching the Grooper Tree Structure by name, type, or ID.
Generate Content Type Dialog Provides a user interface for specifying parameters which control creation of a Grooper Content Type from a CMIS Content Type.
Generate Pattern Dialog Provides a user interface for generating a regular expression from an example of the data to be extracted.
Grooper Administration Options Dialog Provides a user interface for viewing and editing Grooper Administration user options.
Image File Import Dialog Provides a user interface for importing image-based files from a file system through the Scan Control.
IMAP Query Editor Dialog Provides a User Interface for performing IMAP Queries to a Mail Server.
Import Objects Dialog Provides a user interface for selecting the objects to be imported from a Grooper ZIP archive or from a source repository.
LDAP Security Principal Search Dialog Provides a user interface for finding and selecting security principals (i.e. users and groups) from LDAP.
Licensing Contact Info Dialog Provides a user interface for collecting basic contact information.
Licensing Serial Number Dialog Provides a user interface for collecting basic contact information.
Machine Update Results Dialog Provides a user interface for displaying the results of the Update Target Files command on the Machines tab. This includes which machines were affected and which files were updated on each machine.
Manual Deskew Dialog Provides a user interface for manually adjusting the skew angle of an image.
New Folder Dialog Provides a user interface for specifying basic information when a new node is created.
New Item Dialog Provides a user interface for specifying basic information when a new node is created.
Node List Editor Dialog Provides a user interface for allowing a list of Grooper Nodes to be created and managed.
Node Reference Editor Dialog Provides a user reference for selecting a Grooper Node Reference.
Node Type List Editor Dialog Provides a user interface allowing a list of Grooper Node Types to be created and managed.
Object Command Configuration Editor Dialog Provides a user interface allowing the indication of which available Object Commands appear on the Context Menu for Batch View controls in GrooperClientAttended. All commands that apply to Batch Folders and Batch Pages will be available in the editor. Commands that apply to Batch View controls will be limited to those that apply to those Batch Views in use.
Object List Editor Dialog Provides a user interface for managing a list of objects, and their respective properties.
OCR Results Viewer Dialog Provides a user interface for displaying the OCR results for a Batch Object in Layout, Text, or Character View.
OMR Zone Test Dialog Provides a user interface for testing OMR Zones.
Pattern Editor Dialog Provides a user interface for editing of a Data Pattern object.
Placeholder Editor Dialog Provides a user interface for editing placeholders in regular expression output format strings.
Process Activities Dialog Provides multithreaded activity processing on test batches. Allows any unattended task to be applied to items in a test batch. This functionality is useful during the implementation process for testing, as it can apply activities such as Full Text OCR, Classification, or Extraction to batches without the need to push the batch out to production and run it through a Batch Process.

The number of threads can be set in the Grooper Administration Options Dialog.
Property Grid Dialog Provides a user interface displaying a property grid control that allows properties to be configured.
Provider Import Dialog Provides a user interface for executing import operations using an Import Provider.
Publishing Errors Dialog Provides a user interface for displaying a list of errors encountered when publishing a Batch Process.
Reference List Dialog Provides a user reference for displaying a list of references associated with a node in the Grooper Tree Structure.
Registration Zone Test Dialog Provides a user interface for testing Registration Zones.
Repair Branch Dialog Provides a user interface for finding and optionally repairing data errors in a branch of the Grooper Tree Structure.
Reset Steps Dialog Provides a user interface for selecting steps from a Batch Process for which tasks will be reset.
Sample Images Editor Dialog Provides a user interface for editing the list of sample images for an IP Command.
Scanner Condition Dialog Provides a user interface for handling issues encountered during the scan process
Select Allowed Characters Dialog Provides a user interface for selecting a set of characters considered to be valid in terms of OCR by an OCR engine.
Select Batch Process Step Dialog Provides a user interface for selecting a Batch Process Step from a Batch Process.
Select CMIS Folder Dialog Provides a user interface for selecting the CMIS folder object from a CMIS Repository.
Select Document Type Dialog Provides a user interface for selecting a Document Type.
Select Node Types Dialog Provides a user for selecting one or more Grooper Nodes Types.
Select Nodes Dialog
Select Page Dialog Provides a user interface for selecting a Batch Page object from a test batch.
Select Properties Dialog Provides a user interface for selecting one or more properties of an object.
Shell Execute Info Dialog Provides a user interface for editing shell execute information from the Windows Registry.
Validate Branch Dialog Provides a user interface for validating branches of the Grooper tree structure. Executes the property confiuration validation for the selected node and all object below it, producing a consolidated list of validation messages.
Validation Errors Dialog Provides a user interface for displaying validation errors.
Zone Editor Dialog Provides a user interface for editing a zone by dragging a rectangle on a sample page.
