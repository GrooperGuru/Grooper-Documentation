Object Command Configuration Editor Dialog

Provides a user interface allowing the indication of which available Object Commands appear on the Context Menu for Batch View controls in GrooperClientAttended. All commands that apply to Batch Folders and Batch Pages will be available in the editor. Commands that apply to Batch View controls will be limited to those that apply to those Batch Views in use.

Toolbar Commands

Command Name Description
Cancel Cancel the operation and close the dialog.
Disable All Disables all commands.
Enable All Enables all commands.
OK Accept settings and close the dialog.
Reset to Default Reset the selected command back to its default configuration.
