Grooper 2.70 Documentation

Classification Viewer

Displays the contents of a [Batch], along with a list of Classification Candidates.

Child Controls

Control Type Description
[Batch Viewer] Displays the contents of a [Batch] in a hierarchical folder view. The Batch Viewer control is used throughout Grooper to display and edit the contents of a batch.

Context Menu Commands

Command Name Description
Assign Type Opens the [Assign Document Type] dialog and sets the selected [Batch Folder's] document type to the selected [Content Type].

Additional Commands

  Command Name Shortcut Keys Description
Assign Content Type Ctrl+F Assigns a [Content Type] to the currently selected folder.
Combine Loose Pages Ctrl+P Combines each range of contiguous loose pages into a new document, and attempts to classify each.
Go To Next Issue Ctrl+N Sets the selection to the next loose page or un-classified [Batch Folder].
Merge Pages Ctrl+M Inserts the selected pages into a new folder, optionally classified as a content type.
Reprocess Batch Ctrl+F5 Reprocess all or part of the batch through [Classification].
