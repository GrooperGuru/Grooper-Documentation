Grooper 2.70 Documentation

Index Panel Table

A child control used in the IndexPanel to represent a DataTable object.

Context Menu Commands

  Command Name Shortcut Keys Description
Append Ctrl+Ins Append a new row to end of the table.
Clear All Rows Clear all rows from the selected table.
Copy Ctrl+C Places the value of the current cell on the Windows clipboard.
Cut Ctrl+X Places the value of the current cell on the Windows clipboard, and clears the content of the cell.
Delete Row Ctrl+Del Delete the currently selected row.
Duplicate Row Ctrl+D Duplicate the currently selected row. Creates an identical row immediately below the selected row.
Extract Cell Value... F6
Extract Rows...
Insert Ins Insert a row before the current row.
Move Row Down Ctrl+Shift+Down Move the selected row down.
Move Row Up Ctrl+Shift+Up Move the selected row up.
Paste Ctrl+V Pastes a text value from the Windows clipboard to the selected cell.
Rubberband Cell Value F4 Use rubberband OCR to capture a value into the currently selected cell. After activating this command, drag a rectangle around the text to be captured. If the Data Field associated with the cell has a rubberband OCR profile assigned, OCR will be performed on the region. If not, the value will be extracted from full text OCR results, if present. Otherwise, the Default Zonal OCR Profile from the Grooper Root object will be used.
Rubberband Cell Zone F3 Set the highlight zone for the currently selected cell. After selecting this command, drag a rectangle over the region of the page which should be associated with this cell.
Rubberband Rows Use rubberband OCR to capture additional table rows. After activating this command, drag a rectangle around the rows to be captured. If the Data Table has a rubberband OCR profile assigned, OCR will be performed on the region. If not, rows will be extracted from full text OCR results, if present. Otherwise, the Default Zonal OCR Profile from the Grooper Root object will be used.
