Grooper 2.70 Documentation

Batch Manager

Provides a user interface for viewing and managing production Batches.

Child Controls

Control Type Description
Batch List Displays a list of batches.
Batch Info Viewer Provides a tabbed interface for viewing information about a batch.
Batch Task Status Chart Displays a bar chart showing the task processing status for each step in the Batch Process.
Batch Processing History Viewer Displays the history of processes and/or users which have performed task processing on the selected batch.
Batch Event Viewer Displays a list of events which have occurred on a Batch object.
Batch Task Viewer Displays the tasks associated with a Batch.
Batch Details Viewer Displays details about the currently selected batch.

Toolbar Commands

  Command Name Shortcut Keys Description
Batch / Change Display Format / Black White Ctrl+Shift+B Sets the display color format for all pages in the selected batches to black and white.
Batch / Change Display Format / Color Ctrl+Shift+C Sets the display color format for all pages in the selected batches to color.
Batch / Change Display Format / Grayscale Ctrl+Shift+G Sets the display color format for all pages in the selected batches to grayscale.
Batch / New... Create a new Batch. Displays the Batch Properties Dialog to collect information about the new batch.
Batch / Reset / Batch Statistics Resets the stats for the selected batches.
Batch / Reset / Previous Tasks... Deletes previous tasks so that the batch can be reprocessed through steps in the Batch Process. Displays the Reset Steps Dialog to collect information on which steps should be reset.
Options... Display the options screen for the Batch Manager control.
Pause Pauses processing on all selected batches. When a batch is paused, all pending tasks for the batch are deleted and status of the batch is changed to 'Paused'.
Refresh Refreshes the list of batches.
Resume Resume processing on all selected batches. When a batch is resumed, tasks are submitted for all batch elements which have not already been processed through the current step, and the batch status is changed to 'Ready'. To completely reprocess a batch through a step, including all previously processed items, use the Batch / Reset / Previous Tasks command.

Context Menu Commands

  Command Name Shortcut Keys Description
Combine Combine the selected batches into a single batch. All selected batches must have a status of 'Paused' or 'Error'.
Select All Ctrl+A Select all batches in the list.
