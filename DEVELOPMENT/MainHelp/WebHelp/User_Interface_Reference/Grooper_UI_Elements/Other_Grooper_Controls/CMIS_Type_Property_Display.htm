Grooper 2.70 Documentation

CMIS Type Property Display

Displays the properties defined by a CMIS Content Type.

Context Menu Commands

Command Name Description
  Filter / Include All Removes all filtering from the list.
  Filter / Include Inherited Includes inherited properties in the list.
  Filter / Include Non-Inherited Includes non inherited properties in the list.
  Filter / Include ReadOnly Includes readonly properties in the list.
  Filter / Include Writeable Includes writeable properties in the list.
  Grouping / By Inheritance Groups the list of properties by inheritance.
  Grouping / By Updatability Groups the list of properties by updateability.
  Grouping / None Removes all grouping from the list.
