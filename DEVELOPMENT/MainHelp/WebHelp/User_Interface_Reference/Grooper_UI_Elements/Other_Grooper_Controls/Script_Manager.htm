Script Manager

Provides script viewing, compilation, management, and basic editing features.

Child Controls

Control Type Description
Node File Code Editor Provides display and editing of Visual Basic code and project files.

Toolbar Commands

Command Name Description
Actions / Delete Deletes all scripting project files.
Actions / Disable Disables the script from being used by Grooper.
Actions / Update References Updates the references of Grooper libararies to enable compilation in different development environments.
Compile Compiles the script and stores the DLL and PDB to the file store.
Create Creates a scripting project.
  Debug Target Target Grooper Application. Select the Grooper Application in which debugging should occur.
    Choose from the following:
  • Administration
  • Attended Client
  • Unattended Client
Edit in Visual Studio Launches the Visual Studio IDE to edit the script.
  Save Saves the scripting project.
