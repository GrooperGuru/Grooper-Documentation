Grooper 2.70 Documentation

Thumbnail Viewer

Provides a thumbnail-based view of a [Batch].

Toolbar Commands

  Command Name Shortcut Keys Description
  Sort Menu / Sort By Aspect Ratio Ctrl+Shift+A Sort thumbnails from highest aspect ratio to lowest aspect ratio.
  Sort Menu / Sort By Batch Order Ctrl+Shift+B Sort thumbnails as they are ordered in the batch.
  Sort Menu / Sort By Flag Status Ctrl+Shift+F Sort thumbnails so flagged items are first.
  Sort Menu / Sort By Height Ctrl+Shift+H Sort thumbnails from tallest to shortest.
  Sort Menu / Sort By Intensity Ctrl+Shift+I Sort thumbnails from lightest to darkest. One use for this is grouping blank pages together.
  Sort Menu / Sort By Width Ctrl+Shift+W Sort thumbnails from widest to narrowest.
