Grooper 2.70 Documentation

Batch Viewer

Displays the contents of a [Batch] in a hierarchical folder view. The Batch Viewer control is used throughout Grooper to display and edit the contents of a batch.

Context Menu Commands

  Command Name Shortcut Keys Description
Copy Ctrl+C Copies the selected object(s) to the Windows clipboard.
Cut Ctrl+X Cuts the selected object(s) to the Windows clipboard.
Delete Del Deletes the selected object(s).
Move Down Ctrl+Down Move the selected item down.
Move Up Ctrl+Up Move the selected item up.
Paste Ctrl+V Pastes the contents of the Windows clipboard as children of the currenlty selected folder.
Properties... F12 Display the properties for the currenly selected object.

Additional Commands

  Command Name Shortcut Keys Description
Add Selection to New Folder Shift+Ins Creates a new folder and moves all the selected [Batch Pages] inside it.
Adjust Image Ctrl+A Displays a UI allowing the user to edit the display properties of the image. For black and white images, the threshold may be adjusted. For grayscale and color images, the brightness and contrast may be adjusted.
Append to Previous Ctrl+P Append the selected object(s) to the previous folder. If the selected object is a folder, the contents of the selected folder will be appended to the previous folder, and the selected folder will be removed. If the selected object is one or more pages, the pages will be appended to the end of the previous folder.
Crop Ctrl+R Crops the displayed page to the current selection.
Deskew Ctrl+D Deskew Image. This command is provided for the shortcut key.
Goto Next Flagged Item Ctrl+G Moves the focused item to the first flagged page of the batch (if any).
New Folder Ins Adds a new [Batch Folder].
Split Folder Ctrl+S Splits the content of the parent folder at the currently selected node, moving all items from the selected node to the end of the folder into a new folder.
