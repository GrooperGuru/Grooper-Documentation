Grooper 2.70 Documentation

Other Grooper Controls

A list of all Grooper Controls other than Node View Controls, which are a special category.

Control Type Description
ACL Editor Provides a user interface for editing the access control list associated with a node in the Grooper Tree Structure.
Advanced Properties Viewer Displays detailed information about Grooper Node objects, and provides administrative functions for managing them. This NodeView contains multiple tabs for displaying and managing the various aspects of a Grooper Node:
  • General Info: Displays general information about the Node, includings its Unique ID, Database information and Node Attributes.
  • Security: Provides a user interface for managaing access control to the Node.
  • Properties: Displays the properties of the node in Json format.
  • Files: Displays a list of files attached to the node.
  • Values: Displays a list of values associated with the node.
  • References: Displays a list of nodes referenced by the node as well as a list of nodes that this node references.
Batch Details Viewer Displays details about the currently selected batch.
Batch Event Viewer Displays a list of events which have occurred on a Batch object.
Batch Info Viewer Provides a tabbed interface for viewing information about a batch.
Batch List Displays a list of batches.
Batch Manager Provides a user interface for viewing and managing production Batches.
Batch Multi View A container control that hosts one or more child controls derived from BatchView. If more than one child control is to be hosted, then the child controls are automatically added to a TabControl to provide a convenient tabbed UI for switching between the child controls.
Batch Processing History Viewer Displays the history of processes and/or users which have performed task processing on the selected batch.
Scan Viewer Provides document scanning capabilities and displays the contents of a Batch in a hierarchical folder view in a single control. This is a combination control combining the ScanControl and the BatchViewer controls.
Batch Task Status Chart Displays a bar chart showing the task processing status for each step in the Batch Process.
Batch Task Viewer Displays the tasks associated with a Batch.
Thumbnail Viewer Provides a thumbnail-based view of a Batch.
Batch Viewer Displays the contents of a Batch in a hierarchical folder view. The Batch Viewer control is used throughout Grooper to display and edit the contents of a batch.
Billing Items Viewer Provides a user interface for configuring Billing Item Activities.
Classification Tester Allows the testing and training of Classification features.
Classification Viewer Displays the contents of a Batch, along with a list of Classification Candidates.
CMIS Query Builder A UI control for building a CMIS query. Includes a CMIS Query Field List control, along with controls for specifying sort order.
CMIS Query Control Provides a user interface for building a CMIS query string.
CMIS Query Editor Provides user interface for editing a CMIS query string directly. Supports syntax highlighting and code completion.
CMIS Query Field List Displays the list of properties associated with a CMIS Content Type, allowing search criteria to be entered.
CMIS Repository Browser Provides a user interface for browsing the tree structure of a CMIS Repository.
CMIS Repository Searcher Provides a user interface for configuring / testing a CMIS query, and for viewing the results of the query.
CMIS Result Set Viewer Displays a result set from a CMIS query.
CMIS Tree Viewer Displays a branch of a CMIS Repository.
CMIS Type Drop Down Displays a list of CMIS Content Types in a drop-down list.
CMIS Type Property Display Displays the properties defined by a CMIS Content Type.
Code Editor A text editor control supporting syntax highlighting and code completion for various languages.
Collation Pattern Editor Provides a user interface for editing the collation pattern of a Data Type.
CV Classification Model Display Displays classificatin model data for CV-based classification.
Database Table Preview Displays a preview of the table columns and data for the selected database table.
Data Display Timer Displays statistics on a timer, typically while a background process is running.
Data Element Profile Manager Provides a user interface for managing Data Element Profiles for a Content Type.
Data Pattern Editor Provides a control-based user interface for editing a DataPattern object and testing it on objects in a Batch.
Document Viewer Provides display of various non-image file types.
Email Viewer Handles display of RFC822 mail messages.
Extract Document Viewer
Filtered Tree Viewer Displays a branch of Grooper repository, filtering the selections available by type.
Form Type Viewer
Fuzzy Extraction Visualizer When Fuzzy Matching is enabled, this control shows a visualization of the process used to attempt Fuzzy Matching. The top row contains the text that was extracted from the selected Batch Page. The first column contains the text of the selected Vocabulary Part. Green squares represent a potential match and red squares represent the distance of the original text from the defined value. This 'back trace' can be used to calculate Fuzzy Match percentage required to establish a match.
Image Viewer
Index Navigator Displays Document Instance objects, providing a user interface for reviewing and editing the data elements associated with a document. The Index Navigator Panel is a hierarchy of Index Panel Control objects, designed to allow editing of the hierarchical Data Instance objects produced by the Extraction activity.
Index Panel Displays Document Instance objects, providing a user interface for reviewing and editing the data elements associated with a document. The Index Panel is a hierarchy of Index Panel Control objects, designed to allow editing of the hierarchical Data Instance objects produced by the Extraction activity.
Index Panel Document Represents a document in the Index Panel hierarchy.
Index Panel Field A child control used in the IndexPanel to represent a DataField object.
Index Panel Field Grid A child control used in the IndexPanel to represent a DataTable object.
Index Panel Section A child control used in the IndexPanel to represent a Data Section object.
Index Panel Table A child control used in the IndexPanel to represent a DataTable object.
Instance Collection Viewer Displays the Data Instances (values and statistics) for the selected Data Pattern and Batch Object.
Instance Text Viewer Displays the text associated with a Data Instance, optionally showing control characters.
Instance Viewer Displays Data Instance objects, allowing their contents to be exlored.
Ip Results Viewer
License Manager Displays licensing information for a License Server.
Machine Viewer Displays information about a Grooper server or workstation.
Model Display Displays the data associated with a classification model.
Node Drop Down Provides a drop-down browser for the Grooper Tree Structure, allowing a node to be selected.
Node File Code Editor Provides display and editing of Visual Basic code and project files.
Node Property Grid
Node Tree View
Node Tree View
Node View Host
OCR Character Viewer Displays the list of individual characters associated with a set of OCR results.
OCR Layout Viewer Displays OCR results, using HTML to approximate the actual layout.
OCR Results Viewer Displays a tabbed interface for viewing OCR results in layout, text, or character view.
Ocr Text Viewer Displays the text associated with a Data Instance, optionally showing control characters.
Page Classification Tester Test Classification and Separation on a batch.
Page Viewer Displays image-based documents.
Property Grid
Property Mapping Control Provides a user interface for mapping field or property values between objects.
Reference Viewer Displays the incoming and outgoing references for a node.
Report Viewer Renders and displays a Report Instance.
Scan Control Provides document scanning capabilities.
Script Manager Provides script viewing, compilation, management, and basic editing features.
Service Instance Viewer Displays a list of Service Instances for an individual machine, or for all machines attached to the Grooper repository.
Test Batch Source Provides a user interface for selecting and viewing test batches.
Test Image Source
TF-IDF Model Display Displays the data associated with a classification model when Lexical classification is in use.
Tree Editor
Tree Viewer
Type Viewer Displays all of the loaded Type assemblies contained in Grooper, along with their children and attributes.
Vector Comparison Display Displays the Features of a selected document and Document Type and the contribution of those features in Classification as both a Document and Content Type.
