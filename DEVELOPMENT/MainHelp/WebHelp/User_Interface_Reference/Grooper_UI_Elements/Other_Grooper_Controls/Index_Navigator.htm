Index Navigator

Displays [Document Instance] objects, providing a user interface for reviewing and editing the data elements associated with a document. The Index Navigator Panel is a hierarchy of [Index Panel Control] objects, designed to allow editing of the hierarchical [Data Instance] objects produced by the [Extraction] activity.

Reprocess Ctrl+F5 Reprocess documents through extraction.
Property Name Description
Reprocessing Scope Determines which set of documents within the batch will be reprocessed. Can be one of the following values:
  • CurrentDocument: Only the current document will be reprocessed.
  • CurrentToEnd: Reprocess from the current document to the end of the batch.
  • NeedsReview: Reprocess only the items which still need review.
  • EntireBatch: Reprocess the entire batch.
