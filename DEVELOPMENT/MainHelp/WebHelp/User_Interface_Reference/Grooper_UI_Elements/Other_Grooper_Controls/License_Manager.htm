Grooper 2.70 Documentation

License Manager

Displays licensing information for a License Server.

Toolbar Commands

Command Name Description
Activate Online... Use online activition to activate Grooper licensing using serial number. An Internet connection is required for this command to execute.
Check In License Check in the selected license. When critical system errors occur, license may occasianally fail to get checked in. In such cases, this command may be used to manually free the licenses.
Delete License Delete the selected license.
Emergency Activation... Obtain temporary emergency licensing. The emergency activation feature is designed to prevent downtime in situations where unexpected licensing issues occur.
Import License Package... Add new licenses from a Grooper license file (.LIC).
Refresh Refresh the list of licenses.
