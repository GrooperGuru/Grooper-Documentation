Instance Text Viewer

Displays the text associated with a Data Instance, optionally showing control characters.

Context Menu Commands

  Command Name Shortcut Keys Description
Copy Ctrl+C Copies the current selection in the text box to the Clipboard.
Select All Ctrl+A Selects all of the text.
Show Control Characters Ctrl+S Toggles display of control characters.
Word Wrap Ctrl+W Toggles word wrapping.
