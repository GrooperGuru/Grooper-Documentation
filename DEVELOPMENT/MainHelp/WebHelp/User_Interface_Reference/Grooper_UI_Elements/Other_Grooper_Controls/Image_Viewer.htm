Grooper 2.70 Documentation

Image Viewer

Toolbar Commands

Command Name Description
Actual Size Display the current page at 100% resolution.
Fit Width Fit the current page to the width of the display window.
Fit Window Fit the current page to the display window.
Magnify Activate the magnifier mouse tool.
Pan Activate the pan mouse tool. Allows the page to be scrolled by grabbing it with the left mouse button and dragging.
Print... Print the currenlty displayed page.
Rotate Left Rotate the current page 90° counter-clockwise.
Rotate Right Rotate the current page 90° clockwise.
Save... Save the currenlty displayed page as a file.
Select Activate the selection tool.
Select Region Activate the selection mouse tool.
Zoom Activate the zoom mouse tool.
Zoom In Increase the scale factor of the currently displayed page.
Zoom Out Decrease the scale factor of the currently displayed page.

Context Menu Commands

Command Name Description
  Copy To Clipboard / Annotated Image Copies the current page to the Windows clipboard as an image at its current scale factor, including any annotations in the copied image.
  Copy To Clipboard / Full Image Copies the current page to the Windows clipboard as an image at its full resolution.
  Copy To Clipboard / Scaled Image Copies the current page to the Windows clipboard as an image at its current scale factor.
