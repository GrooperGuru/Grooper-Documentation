Grooper 2.70 Documentation

Grooper User Interface Elements

Grooper User Interface Elements include Grooper Controls and Forms. Grooper Controls are grouped into two categories: NodeViews and Other Grooper Controls

Control Type Description
NodeView Controls A list of all Grooper NodeViews. NodeViews are a special type of Grooper Control that gets displayed when a Grooper Node is selected. The Grooper Node Type determines which specific NodeView is displayed.
Other Grooper Controls A list of all other Grooper custom Controls.
Grooper Forms A list of all Grooper custom Forms.
