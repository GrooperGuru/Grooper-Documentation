Grooper 2.70 Documentation

FileStore Utilities NodeView

Displays detailed information about Grooper Node objects, and provides administrative functions for managing them.

Toolbar Commands

Command Name Description
Check for Missing Files Compile a list of Grooper objects that exist in the database but don't have a corresponding file in the File Store.
Check for Orphaned Files Compile a list of Grooper objects that exist in the File Store but not in the Grooper database.
Check FSE Table Validates the File Store Entry database table. Identifies table rows where the Grooper Node Id does not exist in the Grooper system. If the Grooper Node does exist, then that nodes file list will be checked for the existence of the file store entry. Upon completion, the option will be presented to delete identified rows.

Context Menu Commands

  Command Name Shortcut Keys Description
  Delete Selected Files Del Delete the selected file(s) from Grooper.
  Delete Selected Files Del Delete the selected file(s) from Grooper.
  Select All Ctrl+A Select all files in the list.
  View Node Go to the selected Grooper node.
  View Node Go to the selected Grooper node.
