IpElementContainer General NodeView

Provides a user interface allowing Ip Steps in an IP Profile to be managed and allowing the Ip Profile to be tested on test batch images.

Toolbar Commands

Command Name Description
Cancel Cancel the changes.
Delete Delete the selected IP Command.
Diagnostics Mode On Toggles diagnostics mode on or off. By default, IP commands executed in the tester run in diagnostics mode. In this mode, commands generate log information and intermediate images which are useful during configuration and testing, but which are not needed during production image processing. Disabling diagnostics mode will allow commands to execute as they would in production, but no intermediate images or log information will be available.
Execute Execute the IP Profile in the tester.
Move Down Move the selected IP Step down.
Move Up Move the selected IP Step up.
Save Save the changes to the IP Profile.
Save Processed Page Replace the image for the current page with the processed image.
