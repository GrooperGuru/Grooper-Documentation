Grooper 2.70 Documentation

BatchFolder General NodeView

Provides a user interface displaying the properties of a Batch Folder, a BatchViewer for viewing and managing the children of the Batch Folder, and a Page Viewer for displaying the contents of the Batch Folder and its children.

Child Controls

Control Type Description
Node Property Grid
Batch Viewer Displays the contents of a Batch in a hierarchical folder view. The Batch Viewer control is used throughout Grooper to display and edit the contents of a batch.
Page Viewer Displays image-based documents.
Image Viewer

Toolbar Commands

Command Name Description
Cancel Cancel the changes.
Save Save the changes to the Batch Folder.
