Grooper 2.70 Documentation

NodeView Controls

A list of all Grooper NodeView controls. NodeViews are a special type of Grooper Control that gets displayed when a Grooper Node is selected. The selected Grooper Node Type determines which specific NodeView is displayed. Each NodeView generally appears in its own tab in the user interface.

Node Type Tab Name Description
Batch
Batch Batch Viewer Provides a user interface for viewing and managing the children of a Grooper Node.
Batch Batch Provides a user interface displaying the properties of a Batch and a list of Tabs showing various details about the Batch including Processing Status, Processing History, Event Log History, Current Tasks, Activity Statistics, a Batch Viewer and Batch Details.
Batches Folder
Batches Folder Batches Folder Provides a user interface displaying a list of all existing Batches and controls for managing them.
Batch Folder
Batch Folder Batch Folder Provides a user interface displaying the properties of a Batch Folder, a BatchViewer for viewing and managing the children of the Batch Folder, and a Page Viewer for displaying the contents of the Batch Folder and its children.
Batch Folder Index Data Provides a user interface displaying all Index Data associated with the Batch Folder as well as an Image Viewer to display each Data Element. Only applicable if the Batch Folder is assigned a Document Type with an associated Data Model.
Batch Folder Instance Viewer Provides a user interface displaying Data Instances associated with a Batch Folder as well as a view of each data instance image, a text view, and an OCR Results View. Only applicable if the Batch Folder is assigned a Document Type with an associated Data Model.
Batch Page
Batch Page Batch Page Provides a user interface displaying the properties of a Batch Page and a Page Viewer to display the selected Batch Page.
Batch Process
Batch Process Batch Process Provides a user interface displaying the properties of a Batch Process Step as well as properties for the specific Batch Process Activity assigned to the Batch Process Step.
Batch Process Step
Batch Process Step Batch Process Step Provides a user interface for viewing and managing the children of a Grooper Node.
Character Extract Unit Tester Provides a user interface allowing an Unattended Activity to be tested.
Cmis Connection
Cmis Connection Cmis Connection Provides a user interface for connection to a CMIS server, enumerating its repositories, and importing repositories for use with Grooper.
Cmis Content Type
Cmis Content Type Cmis Content Type Provides a user interface displaying the properties of a CMIS Content Type.
Cmis Repository
Cmis Repository Browse Provides a user interface for browing a CMIS Repository.
Cmis Repository Cmis Repository Provides a user interface for configuring and querying a CMIS Repository.
Cmis Repository Search Provides a user interface to search a CMIS Repository.
Content Type
Content Type Classification Testing Provides a user interface for testing Grooper Classification associated with a Content Type.
Content Type Data Element Profiles Provides a user interface for manipulating Data Element Profiles associated with a Content Type, if any.
Content Type Content Type Provides a user interface for displaying the properties of a Content Type and a Data Model preview.
Content Type Separation Testing Provides a user interface for testing Grooper separation associated with a Content Type.
Content Type Weightings Provides a user interface displaying weightings associated with a trained Content Type.
Control Sheet
Control Sheet Control Sheet Provides a user interface displaying the properties of a Control Sheet and a Page Viewer that displays the Control Sheet as the properties are manipulated. The Control Sheet can be sent to a printer or saved to disk from the Page Viewer.
Database Table
Database Table Database Table Provides a user interface displaying column information for the Database Table as well as a preview of data currently existing in the table.
Data Connection
Data Connection Data Connection Provides a user interface to connect to a Sql Server or ODBC Database and allows database tables to be imported for I/O operations in Grooper.
Data Field
Data Field Data Field Provides a user interface displaying the properties of a Data Element as well as an interface for testing the [Data Element] on test Batch documents.
Data Format
Data Format Data Format Provides a user interface displaying a pattern editor and the properties of a Data Format, as well as an interface for testing the Data Format using test batch documents.
Data Model
Data Model Extraction Testing Provides a user interface for testing extraction to a Data Model.
Data Type
Data Type Data Type Provides a user interface displaying the properties of a Data Type as well as an interface for testing the Data Type using test batch documents.
Field Class
Field Class Field Class Provides a user interface displaying the properties of a Field Class as well as an interface for testing the Field Class using test batch documents.
Field Class Weightings Provides a user interface displaying the training weighting information associated with a Field Class.
File Store
File Store Utilities Displays detailed information about Grooper Node objects, and provides administrative functions for managing them.
Form Type
Form Type Form Type Provides a user interface displaying a Form Type.
Grooper Node
Grooper Node Advanced Displays detailed information about Grooper Node objects, and provides administrative functions for managing them. This NodeView contains multiple tabs for displaying and managing the various aspects of a Grooper Node:
  • General Info: Displays general information about the Node, includings its Unique ID, Database information and Node Attributes.
  • Security: Provides a user interface for managaing access control to the Node.
  • Properties: Displays the properties of the node in Json format.
  • Files: Displays a list of files attached to the node.
  • Values: Displays a list of values associated with the node.
  • References: Displays a list of nodes referenced by the node as well as a list of nodes that this node references.
Grooper Node Contents Provides a user interface for viewing and managing the children of a Grooper Node.
Grooper Node * Displays the properties of a Grooper Node.
Grooper Node Scripting Provides script viewing, compilation, management, and basic editing features.
Grooper Root
Grooper Root Events Provides a user interface for administrators to search and view the Grooper event log.
Grooper Root Grooper Root Displays detailed information about Grooper Node objects, and provides administrative functions for managing them.
Grooper Root Licensing Displays detailed information about Grooper Node objects, and provides administrative functions for managing them.
Grooper Root Locks Provides a user interface for administrators to review locked objects, and manually unlock them when needed. Grooper locks objects in the repository during update operations to synchornize the activity of multiple processes or threads. In some cases, such as a database outage or system error, an object may become stuck in locked state. When this occurs, administrators can find and clear these locks using the Lock Manager Control.
Grooper Root Scripts Find and manage all scripts in the Grooper Repository.
Grooper Root Stats Provides a user interface for searching and viewing production statistics.
Grooper Root Types Displays detailed information about Grooper Node objects, and provides administrative functions for managing them.
IPProfile
IPProfile IPProfile Provides a user interface allowing Ip Steps in an IP Profile to be managed and allowing the Ip Profile to be tested on test batch images.
Ip Step
Ip Step Ip Step Provides a user interface displaying the properties of an Ip Step and allowing the Ip Step to be tested on test batch images.
Lexicon
Lexicon Lexicon Provides a user interface for editing the properties and contents of a Lexicon object.
License Server
License Server License Server Provides a user interface allowing a License Server to be configured.
Machine
Machine Machine Provides a user interface displaying the properties of a Machine connected with a Grooper Instance and allowing Grooper Services on the machine to be managed.
Machines Folder
Machines Folder Machines Folder Provides a user interface allowing multiple Machines to be associated with a Grooper Instance for running Grooper Services on.
Machines Folder Services Provides a user interface allowing Grooper Services to be managed on the Machines that have been associated with the Grooper Instance.
Ocr Profile
Ocr Profile Ocr Profile Provides a user interface for configuring an OCR Profile.
Ocr Profile Testing Provides a user interface for testing an OCR Profile.
Page Type
Page Type Page Type Provides a user interface displaying a Training Page and a control for the Training Page to be removed.
Project
Project Project Provides a user interface displaying the properties of a Project and allowing Project Billing Items to be managed.
Report Instance
Report Instance Report Instance Provides a user interface for viewing and managing the children of a Report.
Scanner Profile Folder
Scanner Profile Folder Scanner Profile Folder Provides a user interface allowing a Scanner Profile to be configured.
Separation Profile
Separation Profile Testing Provides a user interface for testing a Separation Profile using test batches.
Miscellaneous
Miscellaneous Attended Activity Tester Provides a user interface allowing an Attended Activity to be tested.
Miscellaneous Redaction Preview Provides a user interface for testing Redaction on test batch documents.
Miscellaneous Unattended Activity Tester Provides a user interface allowing an Unattended Activity to be tested.

* Applies to multiple GrooperNode types. The Tab name is the Node Type Name.
