Grooper 2.70 Documentation

Index Panel Field Grid

A child control used in the IndexPanel to represent a DataTable object.

Context Menu Commands

  Command Name Shortcut Keys Description
Copy Ctrl+C Places the current field value on the Windows clipboard.
Cut Ctrl+X Places the current field value on the Windows clipboard, and clears the field value.
Extract Text... F6
Paste Ctrl+V Pastes a text value from the Windows clipboard to the field.
Rubberband OCR F4 Use rubberband OCR to capture a value into the field. After activating this command, drag a rectangle around the text to be captured. If the Data Field has a rubberband OCR profile assigned, OCR will be performed on the region. If not, the value will be extracted from full text OCR results, if present. Otherwise, the Default Zonal OCR Profile from the Grooper Root object will be used.
Rubberband Zone F3 Set the highlight zone for the field. After selecting this command, drag a rectangle over the region of the page which should be associated with this field.
