Grooper 2.70 Documentation

Scan Viewer

Provides document scanning capabilities and displays the contents of a Batch in a hierarchical folder view in a single control. This is a combination control combining the ScanControl and the BatchViewer controls.

Child Controls

Control Type Description
Scan Control Provides document scanning capabilities.
Node Drop Down Provides a drop-down browser for the Grooper Tree Structure, allowing a node to be selected.
Batch Viewer Displays the contents of a Batch in a hierarchical folder view. The Batch Viewer control is used throughout Grooper to display and edit the contents of a batch.

Additional Commands

  Command Name Shortcut Keys Description
Append Ctrl+Shift+P Starts the scanning process, appending newly-scanned pages to the end of the currently selected folder.
Property Name Description
Scope Specifies how many pages should be scanned. Can be one of the following values:
  • All: Scan all pages in the feeder.
  • Single: Scan only enough pages to replace the current selection.
  • Fixed: Scan a specific number of pages.
Page Count When Scope is set to Fixed, specifies the maximum number of pages to be scanned. A value of 0 will scan all pages in the feeder.
Apply Image Cleanup Alt+I Applies an IP Profile to one or more pages.
Property Name Description
IP Profile The IP Profile to execute.
Insert Ctrl+Shift+I Starts the scanning process, inserting new pages immediately before or after the current selection.
Property Name Description
Insert Position Indicates whether scanned pages should be inserted before or after the current selection. Can be one of the following values:
  • Before: Newly-scanned pages will be inserted before the selected item.
  • After: Newly-scanned pages will be inserted after the selected item.
Scope Specifies how many pages should be scanned. Can be one of the following values:
  • All: Scan all pages in the feeder.
  • Single: Scan only enough pages to replace the current selection.
  • Fixed: Scan a specific number of pages.
Page Count When Scope is set to Fixed, specifies the maximum number of pages to be scanned. A value of 0 will scan all pages in the feeder.
Mark As Reviewed None Marks one or more pages as reviewed. If the page was previously flagged, the flag will be removed.
Replace Ctrl+Shift+R Replace the currently selected pages with the result of a scan operation.
Property Name Description
Scope Specifies how many pages should be scanned. Can be one of the following values:
  • All: Scan all pages in the feeder.
  • Single: Scan only enough pages to replace the current selection.
  • Fixed: Scan a specific number of pages.
  • Selection: Scan only enough pages to replace the current selection.
Page Count When Scope is set to Fixed, specifies the maximum number of pages to be scanned. A value of 0 will scan all pages in the feeder.
Scan Ctrl+Shift+S Starts the scanning process, appending newly-scanned pages to the end of the batch.
Unmark Ctrl+U Marks one or more pages as reviewed. If the page was previously flagged, the flag will be removed.
