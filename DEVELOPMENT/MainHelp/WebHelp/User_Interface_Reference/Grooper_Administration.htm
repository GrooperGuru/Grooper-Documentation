Grooper Administration

Grooper Administration is a browser for the Grooper Tree Structure, used by administrators to configure and manage objects in a Grooper repository. This application uses an explorer-style view, where the tree structure is displayed in a panel on the left, and a panel on the right displays the properties of the currently selected node.

Child Controls

Control Type Description
Tree Editor

Toolbar Commands

Command Name Description
Add Item Add a new item as child of the currently selected node. Items can be Folders or Batches. Displays the New Item Dialog, allowing the user to specify a name for the new item.
Clone Item Clone the currently selected item. Displays the New Item Dialog, allowing the user to specify a name for the cloned item.
Collapse/Expand Hide or display the tree structure in the left pane.
Delete Delete the currently selected item. A confirmation dialog box is displayed to confirm the delete operation.
Refresh Refresh the selected node and all items below it.
Rename Rename the currently selected item.

Main Menu Commands

  Command Name Shortcut Keys Description
Edit / Back Alt+Left Go back to the previously-visited node.
Edit / Clear Branch Attributes... Clear one or more attributes on the selected node and its descendants. Displays the Attributes Editor Dialog, allowing the user to specify which attributes should be cleared.
Edit / Find... Ctrl+F Search the Grooper repository for objects by name, id, or type name. Displays the Find Objects Dialog.
Edit / Forward Alt+Right Go forward in the recently visited nodes list.
Edit / Set Branch Attributes... Set one or more attributes on the selected node and its descendants. Displays the Attributes Editor Dialog, allowing the user to specify which attributes should be set.
File / Change Repository Connect to a different Grooper repository.
  Help / About... Display Grooper version information.
Help / Show Help... Display help.
Tools / Fuzzy Match Visualizer... Displays the Fuzzy Match Visualizer.
Tools / Options... Display the Grooper Administration Options Dialog.
Tools / Repair Branch... Detect and repair errors in the selected node and its descendants. Displays the Repair Branch Dialog.
Tools / Validate Branch... F11 Validates the properties of the selected object and all objects below it, producing a consolidated list of validation messages. Displays the Validate Branch Dialog.
