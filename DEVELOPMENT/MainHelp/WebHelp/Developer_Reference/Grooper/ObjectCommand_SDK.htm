ObjectCommand

Provides the ability to Execute Commands.


Namespace:  Grooper
Inherits from: DynamicTypeDescriptor

Properties

Property Name Property Type Description
CommandContextType Type Represents the type of the Command Object context attribute. If an Object Command is authored for a specific type, that type must be indicated in the Object Commands class context attribute. Indicating a type in this class attribute, will cause the command to be unavailable in all scenarios outside the context.
ShowProperties Boolean During execution, the command's properties will be presented to the user (provided any exist) in a property form/dialog. If for any reason the displaying of this native form should be suppressed, override this property and return true.
