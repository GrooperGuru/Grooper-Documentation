IpCommand

IP commands perform image processing work, such as image cleanup, format conversion, or feature detection. IP Commands are used to construct IP Profiles, and are also used as stand-alone properties on some Grooper types. For example, the Scanner Profile object exposes a Color Detection command in the Color Detection Settings property.

Some IP Commands modify the images they process, such as Line Removal or Auto Border Crop. Other IP Commands such as Sticky Note Detection or Read Barcodes simply detect features in an image, without modifying the image in any way.

Each IP Command defines a set of supported pixel formats. For example, the Threshold command has no relevance to binary (black and white) images, and therefore only includes color and grayscale image formats. When an image is processed by an IP Profile, and command which do not support the pixel format of the image will be skipped.

Properties

Property Name Property Type Description
CommandTypeName String The type of IP command.
EvaluationOperator EvalMethod When multiple conditions are specified, this property controls how they are evaluated. Can be one of the following values:
  • AND: Indicates that all execution conditions must be met in order for this command to execute.
  • OR: Indicates that one or more of the execution conditions must be met in order for this command to execute.
ExecutionConditions List(Of ExecutionCondition) A list of conditions which control whether this command will execute. If no execution conditions are defined, the command will always execute. Execution conditions can be based on properties of the image being processed, or can be based on the output value of a previous command in the list. Only certain types of commands produce output values. For example, the Color Detect command produces an IsColor output value, the Measure Entropy command outputs an Entropy value, and etc.
Name String An optional name for the command. Naming individual commands can help organize a long list of commands, particularly in cases where multiple instances of the same command appear. NOTE: A name is required if the output of the command is to be referenced from an Execution Condition.
SupportedFormats String The pixel formats supported by this command. See property grid in Grooper Administration for supported formats.
SupportedPixelFormats List(Of PixelFormat) Returns a list of supported pixel formats of images that can be processed by the IP Command.
