UnattendedActivity

Serves as the base class for all unattended activities.


Namespace:  Grooper
Inherits from: BatchProcessingActivity

Properties

Property Name Property Type Description
Database GrooperDb Returns the GrooperDB object.
ErrorDisposition IssueDisposition Determines what happens when an error occurs processing an activity. A combination of the following flags:
  • None: The issue will be ignored, and the item will complete successfully.
  • Flag: The associated Batch Folder or Batch Page will be flagged.
  • Log: The issue will be logged to the Grooper log.
  • Stop: The Batch will be set to error.
ExecuteConfirmation String Override this property to specify a string that should be displayed as a confirmation before the Activity is executed. This confirmation will only appear if the Activity is being executed manually as part of a UI. It will not be displayed if the Activity is running outside of the UI.
IsAttended Boolean Returns true if this activity is an attented activity.
MaximumConsecutiveErrors Int32 The maximum number of consecutive errors, after which a critical stop will be raised. A critical stop will cause services to stop running.
Root GrooperRoot Returns the Grooper root object.
StatNames IEnumerable(Of String) Iterates the list of stat names tracked by this activity.
TypeDisplayName String Returns the type display name of this object.
TypeName String Returns the full type name of this object.
