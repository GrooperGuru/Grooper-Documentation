Grooper 2.70 Documentation

Section Profile

Defines settings for a Data Section which apply when processing a specific Content Type.

Inherits from: Data Element Profile

Properties

Property Name Description
General
Override Extract Method
Type: Boolean, Default: False
If set to true, then this profile will override the Extract Method used by the section. Otherwise, the section will be extracted using zonal extraction.
Extract Method
Type: Section Extract Method
Specifies how section instances will be identified and extracted. (See Section Extract Method)
Bounds
Type: Rectangle, Default: (0,0):(0,0)
Defines the rectangular boundary of the section.
Page Index
Type: Int32, Default: 0
The 0-based page index on which the section zone exists.
