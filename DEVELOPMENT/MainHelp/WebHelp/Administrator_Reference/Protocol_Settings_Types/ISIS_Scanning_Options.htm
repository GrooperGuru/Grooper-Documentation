Grooper 2.70 Documentation

ISIS Scanning Options

Defines custom settings for the ISIS Scanner Protocol.

Inherits from: Protocol Settings

Properties

Property Name Description
General
Scan Ahead
Type: Boolean, Default: True
Enables or disables the ISIS Scan Ahead feature. When Scan-Ahead is enabled, pages are buffered in memory as they are scanned, so that the scanner can run at its full rated speed without stopping to wait for the application to process, display, and save each image.
Scan Ahead Maximum Images
Type: Int32, Default: 0
When Scan Ahead is enabled, this property specifies the maximum number of images to scan ahead. A value of 0 indicates unlimited pages.
