MIME Type Handler

MIME type handlers define properties and commands which may be applied to a Batch Folder object with a specific MIME type.

List of MIME Type Handler Types

Name Description
  Mail Message Handles the message/rfc822 Mime Type.
  PDF Document MIME type handler for PDF document types. Defines the PDF-specific properties and commands which may be appied to a Batch Folder object.
  ZIP Archive Mime type handler for Zip Archive files.
