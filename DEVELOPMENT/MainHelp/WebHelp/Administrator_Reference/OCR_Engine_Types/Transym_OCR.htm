Transym OCR

Transym OCR is a commercial OCR engine which provides highly accurate English language OCR.
  • Used for Full Text and Zonal OCR of documents containing machine print characters.
  • Does not support handwriting.
  • Fully installed with Grooper.
  • Requires 1 license per concurrent process.
  • English language only.

Properties

Property Name Description
Image Cleanup
Auto Invert Zones If enabled, the engine will attempt to automatically invert white-on-black zones.
Deskew If enabled, the image will be automatically deskewed prior to OCR.
Deshade If enabled, a deshade image cleanup process will be executed prior to OCR.
Invert Whole Page If enabled, the entire page will be inverted prior to OCR.
Noise Removal If enabled, noise removal will be performed prior to OCR.
Line Removal If enabled, line removal will be performed prior to OCR.
Document Structure
Orientation Indicates the orientation of text to the extracted. Can be one of the following values:
  • Auto: Transym will automatically detect the page orientation.
  • Angle90: Transym will assume that all pages are rotated 90°.
  • Angle180: Transym will assume that all pages are rotated 180°.
  • Angle270: Transym will assume that all pages are rotated 270°.
  • None: Transym will assume that all pages are upright and require no rotation.
Perform Sectioning If enabled, the OCR engine will attempt to detect the structure of the document during OCR processing.
Processing Options
Merge Broken Characters If enabled, broken characters will be merged prior to OCR.
Reject Questionable Lines If enabled, the OCR engine will attempt to detect entire lines of 'junk' and remove them.
Reject Questionable Characters If enabled, individual characters with questionable confidence values will be rejected.
Use Lexicon If enabled, a lexicon will be used to correct words.
Allowed Characters The set of allowed characters for the OCR process.
