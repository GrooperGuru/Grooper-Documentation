Grooper 2.70 Documentation

OCR Engine

OCR Engines extract text from images using optical character recognition. Each OCR Profile in Grooper is configured with an OCR Engine. The OCR Engine selection defines the settings which will be exposed to configure the OCR Profile, and controls the runtime component used for text extraction.

NOTE: Some OCR engines require third-party installations and licensing.

List of OCR Engine Types

Name Description
ABBYY FineReader ABBYY FineReader is a commercial OCR Engine recognized as a leader in the industry, and represents a premium OCR engine offering within Grooper.
  • Used for Full Text OCR of machine print documents or Zonal OCR of machine print and/or handwriting.
  • Performs intensive document structure analysis to determine text flow, making it the best option for generating searchable PDF output.
  • Requires separate installation of ABBY FineReader engine.
  • Requires purchase of add-on FineReader volume licensing.
  • Supports 190 languages.
Grooper Voting A Grooper OCR which performs OCR using multiple OCR Profiles, and then combines the results using a voting mechanism.
  • Perform OCR with multiple different image cleanup configurations.
  • Perform OCR using multiple different OCR engines.
  • Merge and correct results using an intelligent voting mechanism.
  • Requires no additional licensing..
Prime OCR Prime OCR is a commercial voting OCR engine which uses multiple OCR engines and methods to improve OCR accuracy.
  • Used for Full text or Zonal OCR of machine print documents.
  • Does not support handwriting recognition.
  • Requires separate installation of Prime OCR engine.
  • Requires purchase of add-on Prime OCR volume licensing.
  • Supports 11 languages.
Tesseract OCR Tessaract is an open source OCR Engine sponsored by Google.
  • Used for Full Text and Zonal OCR of documents containing machine print characters.
  • Does not support handwriting.
  • Fully installed with Grooper.
  • 100% free - no additional licensing required to use.
  • Supports 60 languages.
Transym OCR 4 Transym OCR is a commercial OCR engine which provides highly accurate English-only OCR.
  • Used for Full Text and Zonal OCR of documents containing machine print characters.
  • Does not support handwriting.
  • Fully installed with Grooper.
  • Requires 1 license per concurrent process.
  • English language only.
Transym OCR 5 Transym OCR 5 is a commercial OCR engine which provides highly accurate multilanguage language OCR.
  • Used for Full Text and Zonal OCR of documents containing machine print characters.
  • Does not support handwriting.
  • Fully installed with Grooper.
  • Requires 1 license per concurrent process.
  • Supports 28 languages.
