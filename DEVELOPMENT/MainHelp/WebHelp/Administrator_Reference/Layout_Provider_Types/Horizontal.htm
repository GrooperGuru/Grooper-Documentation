Grooper 2.70 Documentation

Horizontal

Detects cases where the array elements are arranged in a horizontal row.

Inherits from: Layout Provider

Properties

Property Name Description
General
Maximum Distance
Type: String
Indicates the maximum horizontal distance between elements, in inches.
Alignment
Type: VerticalDataAlignment, Default: Any
Defines the horizontal alignment requirements for elements of a vertical array. A combination of the following flags:
  • Any: Alignment filtering disabled.
  • Top: Only left-aligned results will be returned.
  • Center: Only centered results will be returned.
  • Bottom: Only right-aligned results will be returned.
A value of 'Any' will allow any alignment, so long as the elements overlap on the Y axis.
Alignment Tolerance
Type: String
Specifies the amount of tolerance to allow when analayzing element alignment.
