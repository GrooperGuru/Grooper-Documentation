Grooper 2.70 Documentation

Vertical

Detects cases where the array elements are arranged in a vertical column.

Inherits from: Layout Provider

Properties

Property Name Description
General
Maximum Distance
Type: String
Indicates the maximum vertical distance between elements, in inches.
Alignment
Type: HorizontalDataAlignment, Default: Any
Defines the horizontal alignment requirements for elements of a vertical array. A combination of the following flags:
  • Any: Alignment filtering disabled.
  • Left: Only left-aligned results will be returned.
  • Center: Only centered results will be returned.
  • Right: Only right-aligned results will be returned.
A value of 'Any' will allow any alignment, so long as the elements overlap on the X axis.
Alignment Tolerance
Type: String
Specifies the amount of tolerance to allow when analayzing element alignment.
