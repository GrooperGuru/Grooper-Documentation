License Server
A License Server object is used to store connection information for a running instance of Grooper License Server. That information is made available to Grooper processes which need to check out licenses. While multiple license server connections can be configured in Grooper Administration, all running Grooper processes use the license server designated by the License Server property of the Grooper Root object.

Before configuring a License Server object in Grooper Administration, at least one instance of Grooper License Server needs to be running so that there is a service instance to connect to. Installation of an instance of Grooper License Server can be accomplished using the Services Tab of Grooper Configuration.

Inherits from: Grooper Node

Properties
