Control Sheet
A Control Sheet Object is a special page that can be printed out and inserted into stacks of pages that are to be scanned to dynamically perform various functions within Grooper as the pages are scanned.   When a Control Sheet is scanned, it changes settings within Grooper that affect non-control sheet page(s) scanned following the Control Sheet. For example, a Control Sheet may be created to tell Grooper to create a new Batch, and then document pages scanned afterwards will be part of the new Batch. Or a Control Sheet may be created to tell Grooper to create a new Folder, and then document pages scanned afterwards will be inserted into that Folder. By selectively placing control sheets within a stack of pages to be scanned, scanner operators can manipulate Grooper to perform these common tasks without having to perform any manual configuration within the software.  This gives scanner operators much more control over the scanning process than they would normally have, as the changes in Grooper happen dynamically as the stack of pages is scanning.

Miscellaneous
Name
Type: String
An optional name for the control sheet, to be printed at the top of the control sheet document.
Delete Control Sheet
Type: Boolean, Default: False
If True, the control sheet will be deleted. NOTE: Separation Profile objects have a property named 'Delete All Control Sheets' which will override this setting if enabled.
Single Page Only
Type: Boolean, Default: False
If True, the settings defined on this control sheet will only affect the next page. In cases where the control sheet settings should only affect the page immediately behind the control sheet, this setting removes the necessity to add a control sheet behind the page to change back to the previous settings.
Entry Lines
Written value lines to be added to the control sheet. Prints lines on the control sheet for handwritten information - simply enter the label for each line.
Display Current Date
Type: Boolean, Default: False
If True, the current date will be added to each control sheet.
