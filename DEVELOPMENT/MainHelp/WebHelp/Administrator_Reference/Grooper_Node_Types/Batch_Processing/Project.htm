Grooper 2.70 Documentation

Project
A Project is defined as a collection of billing items, and can be assigned to Batch Processes. Projects are necessary when using the system Project Billing Report. The Project Billing report will calculate a total for each billing item's statistic and the product of each total against the corresponding rate.

Inherits from: Grooper Node
