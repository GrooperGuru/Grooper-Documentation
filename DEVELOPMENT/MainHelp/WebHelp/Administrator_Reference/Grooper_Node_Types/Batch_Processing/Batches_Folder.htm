Batches Folder
A Batches Folder object is a folder which contains Batches. Any folder under the 'Batch Processing â€¢ Batches' node is considered a Batches Folder. This object does not have any configurable properties or functional use other than to display a user interface in Grooper Administration which allows the batches inside the folder to be managed collectively.

Inherits from: Folder

Properties

Property Name Description
General
Allowed Child Types
List of child types which can be stored in this folder. This property controls the list of types displayed when the user clicks 'Add'.
Sorted
Type: Boolean, Default: False
Controls whether children are sorted by name. This setting will take effect the next time an item is added or renamed.
Description
Type: String
Generic property allowing an administrator to document the purpose of this Grooper Node.
