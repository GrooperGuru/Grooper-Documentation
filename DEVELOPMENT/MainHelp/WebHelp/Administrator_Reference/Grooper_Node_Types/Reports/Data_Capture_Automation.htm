Grooper 2.70 Documentation

Data Capture Automation

Displays statistics for Date Extraction and Data Review.

Properties

Property Name Description
Report Settings
Date Range Select a pre-configured date range or select 'Specific Dates' and enter a date range. Can be one of the following values:
  • SpecificDates
  • Today
  • Yesterday
  • MTD
  • YTD
  • Past7Days
  • Past30Days
  • Past60Days
  • Past90Days
Start Date The starting date for the report. This property is only applicable when the Date Range is set to 'Specific Dates'.
End Date The ending date for the report. This property is only applicable when the Date Range is set to 'Specific Dates'.
Batch Process The batch process to run the report for. Leave blank to generate as report for all batch processes. (See Batch Process)
Report Layout Aspect ratio for the Report. Can be one of the following values:
  • Widescreen
  • Letter
General
Description Generic property allowing an administrator to document the purpose of this Grooper Node.

Commands

  Command Name Shortcut Keys Description
Add Multiple Items   Creates multiple items as children of this object.
Clear Children   Deletes all children of this object.
