Grooper 2.70 Documentation

OCR Profile
Defines settings which control how OCR is performed.

Inherits from: Grooper Node

Properties

Property Name Description
General
OCR Engine
Type: String
The OCR Engine to use with this OCR profile. Please note that some OCR engines require additional licensing. See OCR Engine Types for details on the OCR engines available in Grooper.
IP Profile
Type: IP Profile
The IP Profile to be used for temporary image cleanup prior to OCR. Does not permanently alter the page images. Do not include any commands which change the size or resolution of the image. (See IP Profile)
Region of Interest
Type: Rectangle, Default: (0,0):(0,0)
Indicates the region of the page to be processed.
Description
Type: String
Generic property allowing an administrator to document the purpose of this Grooper Node.
Synthesis
Synthesis Method
Type: SynthesisMethodEnum, Default: Full
The method by which (x,y) character locations will be converted to a text flow. Can be one of the following values:
  • None: Grooper will perform no synthesis. The text flow will reflect what was generated by the OCR engine.
  • Full: Grooper will re-synthesize results received from the OCR engine.
Font Pitch
Type: FontPitchMode, Default: Auto
When synthesis is enabled, this setting controls how the font pitch is detected. Can be one of the following values:
  • Auto: Indicates that Grooper should automatically determine the font pitch for each segment of text.
  • Fixed: Indicates that all text should be treated as fixed-pitch.
  • Variable: Indicates that all text should be treated as variable-pitch.
Maximum Variance
Type: Double, Default: 10%, Range: 0% - 100%
When font pitch detection is enabled, the maximum variance in character cell sizes allowed for a fixed-pitch font. By definition, a fixed-pitch font is one where all characters occupy the exact same amount of horizontal space. Due to the imperfect nature of scanned document images, there will be some in the widths of character cells. This value controls how much variance is allowed before the segment is determined to be a variable pitch font.
Segment End Ratio
Type: Double, Default: 125%, Range: 0% - 500%
When synthesis is enabled, controls how wide a gap must be in relation to the current font size in order to constitute the end of a segment.
Segment Reprocessing Threshold
Type: Double, Default: 0%, Range: 0% - 100%
Text segments with an average character confidence below this value will re-processed through OCR. A value of 0 disables segment reprocessing entirely.
Iterative Processing
OCR Iterations
Type: Int32, Default: 1
Indicates the number of iterations for iterative OCR. A setting of 1 indicates that iterative OCR is disabled. Iterative OCR improves accuracy by performing OCR in multiple passes. On each successive pass, characters recognized in previous passes are digitally dropped out of the image.
Enable Cell Validation
Type: Boolean, Default: False
Indicates whether cell validation should be used. When cell validation is enabled, after OCR is performed on the full image, the image will be divided into an NxN matrix of cells, specified by the Rows and Columns properties. OCR is then performed on each cell independently, and the results are merged with the full page OCR. Edge characters within 0.01" the cell edges are automatically deleted.
Rows
Type: Int32, Default: 2
When cell validation is enabled, the number of vertical cells the image should be divided into.
Columns
Type: Int32, Default: 2
When cell validation is enabled, the number of horizontal cells the image should be divided into.
Cell Edge Buffer
Type: Double, Default: 0.1, Range: 0.05 - 0.25
The size, in inches, of a buffer zone around the border of the cell. Any characters overlapping the buffer zone will be eliminated. This helps eliminate partial characters which may be picked up on cell boundaries.
Cell Overlap
Type: Double, Default: 0.25, Range: 0 - 2
When cell validation is enabled, the amount, in inches, that cells will overlap each other.
Skip First Column
Type: Boolean, Default: False
When Cell Validation is in use, indicates whether the first column of cells should be skipped. Speeds up the cell validation process by skipping the first column of cells. Most of the issues addressed by cell validation tend to occur when graphical elements appear to the left of the text. As such, re-OCRing the first column of cells tends to add limited benefit in some cases.
Results Filtering
Minimum Character Confidence
Type: Double, Default: 0%, Range: 0% - 100%
Any characters with a confidence below this value will be discarded.
Eliminate Edge Characters
Type: Boolean, Default: False
If enabled, characters near the edge of the ROI will be eliminated based on the 'Minimum Distance to Edge' property.
Minimum Distance to Edge
Type: Double, Default: 0.01
When Eliminate Edge Characters is enabled, this specifies the minimum distance, in inches, between a character and the edge of the ROI. Any character closer than this will be eliminated from the OCR output.
Eliminate Isolated Symbols
Type: Boolean, Default: False
If enabled, segments of one or more symbols which are not part of a text line will be eliminated. The Maximum Symbol Ratio property controls what percentage of the segment must be symbols.
Maximum Symbol Ratio
Type: Double, Default: 1
When Eliminate Isolated Symbols is enabled, this specifies the minimum ratio of symbol characters a segment must contain in order for it to be eliminated from the OCR output. A value of 1 will require that every character in the segment be a symbol.
Minimum Character Size
Type: Size Ex F, Default: (0,0)
The minimum absolute character size, in inches. Characters smaller than this will be eliminated from the OCR output. A value of 0 for Width or Height indicates that there is no minimum."
Maximum Character Size
Type: Size Ex F, Default: (0,0)
The maximum absolute character size, in inches. Characters Larger than this will be eliminated from the OCR output. A value of 0 for Width or Height indicates that there is no maximum.
Minimum Font Size
Type: Double, Default: 0
The minimum font size, in points. Any characters which are measurable and have a font size smaller than the indicated point size will be eliminated from the OCR output. A value of 0 indicates that there is no minimum. (1 point = 1/72")
Maximum Font Size
Type: Double, Default: 0
The maximum font size, in points. Any characters which are measurable and have a font size larger than the indicated point size will be eliminated from the OCR output. A value of 0 indicates that there is no maximum. (1 point = 1/72")
