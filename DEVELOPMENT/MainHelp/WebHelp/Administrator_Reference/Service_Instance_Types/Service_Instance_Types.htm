Grooper 2.70 Documentation

Service Instance

Service Instances represent processes which may be installed as Grooper services, such as the Grooper Licensing service or the Task Processing service. Service instances must be installed from Grooper Configuration, using the Service Manager Control on the "Services" tab. After initial installation, service instances may be started, stopped, or configured using the Service Instance Viewer control in Grooper Administration. The Service Instance Viewer can be accessed from the property panel of any Machine object in Grooper. To manage all services for an entire Grooper repository, use the Service Instance Viewer found on the Machines Folder (immediately below the Infrastructure Folder).

List of Service Instance Types

Name Description
ApplicationXtender CMIS Server Exposes an EMC ApplicationXtender repository over CMIS. For a specific AX data source, exposes each of the AX applications as a CMIS repository. Each repository has a single content type with properties matching the field layout of the corresponding AX application.
Grooper CMIS Server Provides web-based connectivity to a Grooper repository over the CMIS protocol.
Grooper Content Manager CMIS Server Exposes a Grooper Content Manager repository over CMIS. Each Grooper Content Manager project corresponds to a CMIS repository.
Grooper Licensing Distributes runtime licenses to Grooper applications. An instance of Grooper Licensing is required in all deployment scenarios with the exception of a stand-alone installation. In a stand-alone system, simply install licenses into the pre-installed 'Local' License Server object in Grooper Administration.

To install licensing in a client-server environment, the following steps must be completed:
  • Using Grooper Configuration, install an instance of Grooper Licensing and start it. Remember the host name and port number on which the service is running.
  • Using Grooper Administration, create a new License Server object and configure it with the host name and port number of the service.
  • Use the 'Activate Online' command of the License Server object, and enter the appropriate Grooper serial number.
  • The view should refresh to display the newly-installed licenses.
Import Watcher Import Watcher is a service which polls an Import Provider for new documents, and imports them based on rules.
License Renewal Service Performs usage reporting when the Grooper pay-as-you-go licensing model is in use.
System Maintenance Service Provides Grooper system maintenance services. A timer service that clears Log and Stat entries older than the specified retention periods.
Task Processing The Task Processing service executes Unattended Activities, which perform the work associated with a step in a Batch Process.
