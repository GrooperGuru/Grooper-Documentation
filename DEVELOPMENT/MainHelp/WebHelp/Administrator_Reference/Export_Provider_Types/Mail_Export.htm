Grooper 2.70 Documentation

Mail Export
Provides export connectivity to IMAP mail servers. Only documents with a MIME type of message/rfc822 may be exported using this provider.

Inherits from: Export Provider

Properties

Property Name Description
Mail Server Connection
Mail Server
Type: String
The host name or IP address of an IMAP mail server.
Use SSL
Type: Boolean, Default: True
If true, the connection to the Mail Server will be established using SSL.
Port No
Type: Int32, Default: 993
The port number on which the IMAP server is running. The default port number for an SSL-enabled IMAP server is 993. With SSL disabled, the default port is 143.
User Name
Type: String
The user name to be used for logging in to the Mail Server. Note: The domain name may be required to access mailboxes. e.g. bis\grooperdemo.
Password
Type: String
The password to be used for logging in to the Mail Server.
Alternate Mailbox Name
Type: String
Specifies the name of a mailbox other than the login user's default mailbox. If this value is blank, the user's default mailbox will be opened. Note that not all IMAP server implementations support multiple mailbox access.
General
Export to Original Path
Type: Boolean, Default: False
If true, then export the mail messages to the original import path. The Batch Folder object must have an existing Mail Link to use this feature.
Target Mail Folder
Type: String
The mail folder to which content will be exported.
Update Link
Type: Boolean, Default: True
If true, the Content Link on the document will be updated to point at the exported content.
