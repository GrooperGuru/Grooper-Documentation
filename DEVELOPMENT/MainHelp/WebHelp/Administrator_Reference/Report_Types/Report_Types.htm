Grooper 2.70 Documentation

Report

Defines a Grooper Report. Can be used to author Reports by providing a Report Definition Language Client file as a Stream, Data Source(s) for the rdlc resource, and optional Report Parameters. For convenience, the base class GetScriptResource method can be used to return the .Rdlc report class by name.

List of Report Types

Name Description
System Reports
Batch StatisticsAggregates all Processing Activity statistics for a Batch Process.
Data Capture AutomationDisplays statistics for the Date Extraction and Data Review Attended Activities.
Data Review OperatorsDisplays the top Data Review operators for Records Viewed, Records Edited or Keystrokes for a given date range.
KeystrokesProvides Keystroke quantities for all published Batch Processes, or for a single Batch Process.
Pages ScannedDisplays a bar chart indicating the quantity of pages scanned by User, Machine or Batch Process.
Production BatchesDisplays the quantity of Production Batches grouped by Batch Process, Processing Activity, Processing Step or by Batch Status.
Project BillingCalculates Billing Item quantities and optional dollar amounts for a configured Project and date range.
Query ReportRepresents a Tabular Grooper Report based on data returned by an SQL Query.
Scan Operator EfficiencyReports Scan Operator statistics.
Scan VolumeDisplays Scanned Page quantities. Results can be filtered by Date Range and by Batch Process.
