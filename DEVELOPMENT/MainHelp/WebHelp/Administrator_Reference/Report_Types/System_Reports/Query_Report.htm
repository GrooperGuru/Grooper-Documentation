Grooper 2.70 Documentation

Query Report

Represents a Tabular Grooper Report based on data returned by an SQL Query.

Inherits from: Table Report

Properties

Property Name Description
Report Settings
Database Connection
Type: Data Connection
Data Connection to be used to execute the Report Query. (See Data Connection)
Query Statement
Type: String
Database Query returning the data to display in the Report.
Appearance
Page Layout
Type: LayoutType, Default: Widescreen
Page Size for the report. Can be one of the following values:
  • Widescreen
  • Letter
  • Landscape
Set to Letter or Landscape if printing is intended. Set to Widescreen if displaying on a monitor is intended.
Column Widths
Type: String
Optional comma-delimited list of column widths in inches. For example, to indicate 2 inches for the first column and 1/2 inch for the second column, enter the following value:
2, 0.5
Column Formats
Type: String
Optional comma-delimited list of Fomat Specifiers. For example, to indicate Currency to two decimal places for the third column, enter the following value:
"","",C2

See the following articles for detailed Format Specifier information:
  • Standard Numeric
  • Standard Date and Time
Header Font Size
Type: Int32, Default: 10, Range: 1 - 200
Font Size in points for the column header cells.
Data Font Size
Type: Int32, Default: 10, Range: 1 - 200
Font Size in points for the table data cells.
Center Table
Type: Boolean, Default: False
If this property is set to True, the table will be centered horizontally on the report page.
Header Row Background Color
Background color for the Header Row.
Data Row Background Color
Background color for the Data Rows.
