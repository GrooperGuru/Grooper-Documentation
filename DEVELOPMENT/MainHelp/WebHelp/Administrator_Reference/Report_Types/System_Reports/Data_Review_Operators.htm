Grooper 2.70 Documentation

Data Review Operators

Displays the top Data Review operators for Records Viewed, Records Edited or Keystrokes for a given date range.

Inherits from: Bar Chart Report

Properties

Property Name Description
Report Settings
Statistic
Type: StatType, Default: Keystrokes
The statistic for which quantities will be aggregated. Can be one of the following values:
  • Keystrokes
  • RecordsViewed
  • RecordsEdited
Date Range
Type: Date Range, Default: Month to Date
 
Appearance
Report Layout
Type: LayoutType, Default: Widescreen
Page Size for the report. Can be one of the following values:
  • Widescreen
  • Letter
  • Landscape
Set to Letter or Landscape if printing is intended. Set to Widescreen if displaying on a monitor is intended.
