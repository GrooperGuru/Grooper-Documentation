Event-Based Separation

Separate the batch based on Separation Events. Separation events can separate based on patch codes, barcodes, blank pages, detected shapes, or page content types.

Inherits from: Control Sheet Separation

Properties

Property Name Description
General
Events
Defines the events which will trigger separation.
Enable Control Sheets
Type: Boolean, Default: False
Enables control sheet detection and processing.
Enable Page Classification
Type: Boolean, Default: False
Enables use of real time visual classification.
Content Model
Type: Content Model
(See Content Model)
