Grooper 2.70 Documentation

Scanner Settings



Inherits from: Connected Object

Properties

Property Name Description
Basic Settings
Resolution
Type: Int32, Default: 0
The resolution at which images will be scanned.
Color Format
Type: ColorFormat, Default: Color
The color format in which images will be captured. Can be one of the following values:
  • Color
  • Grayscale
  • Bitonal
Page Size Mode
Type: PageSizeMode, Default: Auto
The method used to determine the page size. Can be one of the following values:
  • Auto
  • Standard
  • Custom
Custom Page Size
Type: Rectangle, Default: (0,0):(0,0)
Enter a custom page size, expressed as the area to be scanned. For example, to scan a 24\" x 24\" page in its entirety, enter (0,0)-(24,24). To scan the same page but eliminate a 1\" border, enter (1,1)-(22,22).
Standard Page Size
Select a standard page size from the list. If the page size does not appear in this list, change the Page Size Mode to Custom and then enter a Custom Page Size.
Turbo Mode
Type: Boolean, Default: False
Speeds up the capture process by sampling a lower resolution in the Y direction.
Scanning Speed
Type: ScanSpeed, Default: 0
Controls how fast the paper transport moves. This setting does not affect image quality, but merely allows the speed of the transport to be adjusted to accommodate paper condition. Can be one of the following values:
  • Fast
  • Medium
  • Slow
  • Slowest
Thresholding
Threshold Method
Type: Threshold
When the Color Format is set to Bitonal, this setting controls the method by which images are converted to black and white. (See Threshold)
Color Management
Brightness
Type: Int32, Default: 0
The brightness adjustment to be applied during scanning.
White Point
Type: Int32, Default: 0
The white point to be applied during scanning.
Black Point
Type: Int32, Default: 0
The black point to be applied during scanning.
Gamma
Type: Double, Default: 0
The gamma adjustment to be applied during scanning.
Paper Handling
Paper End Action
Type: PaperEndCondition, Default: Eject
Controls what action is taken after each page is scanned. Can be one of the following values:
  • Eject
  • DoNothing
  • Rewind
  • PaperLength
  • RewindAndStage
Paper Justification
Type: PaperJustification, Default: Side
Indicates whether paper will be fed into the scanner centered, or aligned to the side of the transport. Can be one of the following values:
  • Side
  • Center
