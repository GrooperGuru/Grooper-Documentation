Grooper 2.70 Documentation

Mail Link
Defines a reference to an email on an IMAP mail server.

Inherits from: [Content Link]

Properties

Property Name Description
Mail Server Connection
Mail Server
Type: [String]
The host name or IP address of an IMAP mail server.
User Name
Type: [String]
The user name to be used for logging in to the Mail Server. Note: The domain name may be required to access mailboxes. e.g. bis\grooperdemo.
Use SSL
Type: [Boolean], Default: True
If true, the connection to the Mail Server will be established using SSL.
Port No
Type: [Int32], Default: 993
The port number on which the IMAP server is running. The default port number for an SSL-enabled IMAP server is 993. With SSL disabled, the default port is 143.
Mail Item Info
Mailbox Name
Type: [String]
The mailbox name in which the mail item exists. If none is specified, the default mailbox associated with login user is assumed.
Folder Name
Type: [String]
The folder in which the mail item exists.
Message ID
Type: [Int32], Default: 0
The unique ID of the mail item.
Linked MailMessage Info
Mailbox Account Name
Type: [String]
Returns the name of the mailbox account in which the message is stored.
Date Sent
Type: [String]
The date the mail message was sent.
From
Type: [String]
The sender of the mail message.
To
Type: [String]
The recipients of the mail message.
CC
Type: [String]
The carbon copy recipients of the mail message.
Subject
Type: [String]
The subject of the mail message.
Content Link Standard Values
Full Path
Type: [String]
The full path to the linked object.
Object Name
Type: [String]
The name of the linked object.
Path
Type: [String]
The path to the linked object, minus the name.
General
Error Message
Type: [String]
If set, indicates an error which occurred while attempting to access the remote object.
