Grooper 2.70 Documentation

Append
Starts the scanning process, appending newly-scanned pages to the end of the currently selected folder.

Inherits from: ControlCommand(Of Batch Folder)

Properties

Property Name Description
General
Scope
Type: FeederScope, Default: All
Specifies how many pages should be scanned. Can be one of the following values:
  • All: Scan all pages in the feeder.
  • Single: Scan a single page only.
  • Fixed: Scan a specific number of pages.
Page Count
Type: Int32, Default: 0
When Scope is set to Fixed, specifies the maximum number of pages to be scanned. A value of 0 will scan all pages in the feeder.
