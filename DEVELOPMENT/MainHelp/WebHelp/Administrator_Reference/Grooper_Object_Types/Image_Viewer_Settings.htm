Grooper 2.70 Documentation

Image Viewer Settings

Defines settings for the PageViewer control.

Inherits from: Grooper Object

Properties

Property Name Description
General
Anti-Aliasing Mode
Type: AntiAliasMode, Default: Full
Defines the anti-aliasing mode for image display. Can be one of the following values:
  • None
  • ScaleToGray
  • ReductionOnly
  • Full
Smoothing Mode
Type: SmoothingMode, Default: HighQuality
Defines the smoothing mode used for annotation display. Can be one of the following values:
  • Default: Specifies no antialiasing.
  • HighSpeed: Specifies no antialiasing.
  • HighQuality: Specifies antialiased rendering.
  • None: Specifies no antialiasing.
  • AntiAlias: Specifies antialiased rendering.
  • Invalid: Specifies an invalid mode.
Hide Annotations
Type: Boolean, Default: False
Defines the smoothing mode used for annotation display.
