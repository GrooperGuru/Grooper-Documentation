Grooper 2.70 Documentation

Auto Border Invert
Detects dark borders around the edge of an image and inverts or fills the area occupied by the border.

Inherits from: Border Detect

Properties

Property Name Description
General
Operation
Type: OperationType, Default: InvertFillSides
Specifies how the detected border should be removed. Can be one of the following values:
  • Fill: The detected border area will be filled with solid white.
  • Invert: The detected border area will be inverted.
  • InvertFillSides: The detected border area will be inverted, except that the left and right sides will be filled with solid white.
Border Region Size
Type: Unit Border, Default: 0.25in
Defines the maximum size of a border region on each side of the image. Any detected border which exceeds this limit will be ignored. Review the "Zoning" output image to see the resulting region overlayed on the image in red.
Minimum Border Weight
Type: Double, Default: 90%, Range: 0% - 100%
The fill percentage required for border detection. A value of 100% will require a perfectly solid border. Lower values allow increasing imperfection in the border.
Minimum Border Thickness
Type: Unit Border, Default: 1pt
The minimum thickness of an edge artifact which will be considered a border.
Maximum Border Inset
Type: Unit Border, Default: 0.05in
Defines the maximum distance from the edge of the image for a detected border. Review the "Zoning" output image to see the resulting region overlayed on the image in cyan.
Post Processing
Remove Corner Artifacts
Type: Boolean, Default: True
If set to true, artifacts left behind in the corners of the border will be removed. This setting is primarily designed to handle borders with rounded corners.
Edge Cleanup Radius
Type: String, Default: 2px
Defines how aggressively artifacts on the edge of the border will be removed. An empty or zero value will disable edge cleanup.
Border Adjust
Type: Unit Border
The amount by which to expand the border on each side. Positive values will increase the thickness of the detected border, and negative values will decrease it.
Preprocessing
Binarization
Type: Binarize
If enabled, border detection will be performed on a binarized version of the input image. Otherwise, border detection will be performed on the original image. The objective in tuning binarization settings is to produce an image where the border is black, and the image background is white. (See Binarize)
Command Info
Command Type
Type: String
The type of IP command.
Supported Pixel Formats
Type: String
The pixel formats supported by this command.
