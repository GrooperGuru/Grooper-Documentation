Grooper 2.70 Documentation

Crop
Crops borders of an image by a specified amount.

Inherits from: IP Command

Properties

Property Name Description
General
Method
Type: CropMethod, Default: Absolute
The amount of space, in inches, to crop from the left edge of the image. Can be one of the following values:
  • Absolute: The image will be cropped using an absolute region.
  • FromBorders: The image will be cropped by a specific amount on each side.
Crop Region
Type: Rectangle, Default: (0,0):(0,0)
The region to which the image should be cropped
Border Size
Type: Border, Default: 0
The amount of space, in inches, to crop from each edge of the image.
Command Info
Command Type
Type: String
The type of IP command.
Supported Pixel Formats
Type: String
The pixel formats supported by this command.
