Region Detector



Inherits from: Grooper Object

Properties

Property Name Description
General
Minimum Size
Type: Unit Size, Default: 8pt
The minimum size of an inverted text region. Blobs on the Candidates image which are smaller than this will not be further analyzed.
Minimum Area
Type: String, Default: 0.25in
The minimum area of a detected region. For example, a value of 0.25in requires a minimum area of 0.25inÂ². Each blob on the Candidates image which meets the Minimum Size requirement must also have a total area equal to or greater than this value.
Minimum Fill
Type: Double, Default: 50%, Range: 0% - 100%
Specifies the minimum fill percentage of a region. This value is used to filter the obects on the Candidates image by how closely they resemble a rectangle. A value of 100% will require each region on the Candidates image to be a perfectly solid rectangle. A value of 0% will drop out any arrangement of specks which meets the size limits.
Connectivity
Type: MarkerConnectivity, Default: Connect8
Specifies the pixel connectivity requirements for region detection. Can be one of the following values:
  • Connect8: Specifies 8-connectivity, where pixel connections may be horizonal, vertical, or diagonal.
    111
    1X1
    111
  • Connect4: Specifies 4-connectivity, where pixel connections may be horizontal or vertical.
    010
    1X1
    010
Zone Adjustment
Type: String
The distance by which to adjust all sides of each detected region. Positive values will expand the region, and negative values will shrink it.
