Grooper 2.70 Documentation

Tesseract OCR Settings



Inherits from: Ocr Engine Settings

Properties

Property Name Description
Processing Options
OCR Engine Mode
Type: EngineMode, Default: TesseractOnly
Specifies the OCR engine mode: TesseractOnly - Normal paragraph extraction. CubeOnly - Extracts the minimal number of paragraphs (either one paragraph per block or only paragraphs which start with a dropped capital). TesseractAndCube - This setting is deprecated and will be removed in future versions. Each line is extracted to a separate paragraph formatted with spaces. Default - Each line is extracted to a separate paragraph without space formatting, blank spaces are to separate words only. Can be one of the following values:
  • TesseractOnly
  • CubeOnly
  • TesseractAndCube
  • Default
OCR Language
Type: OcrLanguage, Default: English
Specifies the language of the text to Ocr. Can be one of the following values:
  • English
Page Segmentation Mode
Type: PageSegMode, Default: OsdOnly
Specifies the page segmentation mode. Can be one of the following values:
  • OsdOnly: Orientation and script detection (OSD) only.
  • AutoOsd: Automatic page segmentation with orientation and script detection (OSD).
  • AutoOnly: Automatic page segmentation, but no OSD, or OCR.
  • Auto: Fully automatic page segmentation, but no OSD.
  • SingleColumn: Assume a single column of text of variable sizes.
  • SingleBlockVertText: Assume a single uniform block of vertically aligned text.
  • SingleBlock: Assume a single uniform block of text.
  • SingleLine: Treat the image as a single text line.
  • SingleWord: Treat the image as a single word.
  • CircleWord: Treat the image as a single word in a circle.
  • SingleChar: Treat the image as a single character.
  • Count: Number of enum entries.
Allowed Characters
Type: String
The set of allowed characters for the OCR process.
Image Default Resolution
Type: Int32, Default: 0
Default image resolution dpi.
Merge Broken Characters
Type: Boolean, Default: False
Merge suspect partitions.
Image Cleanup
Vigorously remove Noise
Type: Boolean, Default: False
Vigorously remove noise.
Do not remove Noise Blobs
Type: Boolean, Default: False
Don't remove noise blobs.
Miscellaneous
Unrecognized Character
Output character for unidentified blobs.
Vertical Detection
Type: Boolean, Default: False
Enable vertical detection.
Vertical Text Page Mode Force
Type: Boolean, Default: False
Force using vertical text page mode.
Vertical Textlines Fraction
Type: Double, Default: 0
Fraction of textlines deemed vertical to use vertical page mode.
