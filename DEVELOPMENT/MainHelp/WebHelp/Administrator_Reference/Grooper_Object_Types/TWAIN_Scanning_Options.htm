Grooper 2.70 Documentation

TWAIN Scanning Options

Defines custom settings for the TWAIN Scanning Protocol.

Inherits from: Protocol Settings

Properties

Property Name Description
Options
Hide Interface
Type: Boolean, Default: False
If set to true, the scanner driver's user interface will be hidden during scanning. Note that some scanner drivers do not support hiding the user interface. In such cases, this setting will be disabled.
Enable Compression
Type: Boolean, Default: False
If set to true, the scanner driver will return compressed images.
Compression Mode
Type: CompressionMode, Default: None
The compression mode to use for image transfer. In some cases, performance can be improved by using compression during image transfer. Please note that the list of supported compression modes may change when the color format is changed in the driver settings, leaving the profile configured with an invalid compression type. Can be one of the following values:
  • None
  • BitFields
  • Group3
  • Group4
  • Jbig
  • Jpeg
  • Lzw
  • Packbits
  • Png
  • Rle4
  • Rle8
Device Info
Product Name
Type: String
The name of the scanner device.
Driver TWAIN Version
Type: String
The TWAIN protocol version implemented in the scanner driver.
Manufacturer
Type: String
The manufacturer of the capture device.
Serial Number
Type: String
The serial number of the scanner, if one is provided by the driver.
