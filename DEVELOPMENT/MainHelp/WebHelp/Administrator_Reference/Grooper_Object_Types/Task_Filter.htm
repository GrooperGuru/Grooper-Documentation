Grooper 2.70 Documentation

Task Filter



Inherits from: Connected Object

Properties

Property Name Description
Task Filtering
Batch
Type: Batch
If a Batch is indicated, then tasks will only be processed for that batch. (See Batch)
Batch Process
Type: Batch Process
If a Batch Process is indicated, then tasks will only be processed for that batch process. (See Batch Process)
Batch Process Step Name
Type: String
If a Batch Process Step Name is indicated, then Tasks will only be processed for Batch Process Steps that have that specific name. If more than one Batch Process has the specified Batch Process Step name, then tasks from any of those Batch Processes at the steps with the specified name could be returned by the task filter. To specify a Step within a specific Batch Process, select a specific Batch Process in addition to the Batch Process Step Name.
Activity Type
Type: String
If an Activity Type is indicated, then tasks will only be processed for that Activity.
