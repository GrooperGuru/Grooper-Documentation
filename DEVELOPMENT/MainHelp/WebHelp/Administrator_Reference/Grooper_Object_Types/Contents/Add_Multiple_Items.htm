Grooper 2.70 Documentation

Add Multiple Items
Creates multiple items as children of the selected object.

Inherits from: UICommand(Of Grooper Node)

Properties

Property Name Description
General
Item Type
Type: String
The type of item to create. One child item of this type will be created for each name specified in the Item Names property.
Item Names
Type: String
The names of the child items to create. For each name provided in the list, a child item of the specified Item Type will be created.
