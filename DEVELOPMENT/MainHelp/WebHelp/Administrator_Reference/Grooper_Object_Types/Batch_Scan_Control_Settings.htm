Grooper 2.70 Documentation

Batch Scan Control Settings



Inherits from: Control Settings

Properties

Property Name Description
Profiles
Scanner Profile
Type: Scanner Profile
If set, the operator will be restricted to the specified Scanner Profile, and will be unable to edit scanner settings. If no Scanner Profile is specified, the user will be allowed to select scanner profiles and edit settings. (See Scanner Profile)
IP Profile
Type: IP Profile
The IP Profile to be applied in real time during scanning. Use this property to apply permanent image cleanup in real time during scanning. (See IP Profile)
Separation Profile
Type: Separation Profile
The Separation Profile to be applied in real time during scanning. NOTE: The separation profile selected must use a Separation Provider which supports real-time separation. In general, Separation Providers which depend on OCR do not support real-time operation, and must be executed from the Separation activity. (See Separation Profile)
General
Enable Undo
Type: Boolean, Default: False
Causes an Undo Image to be saved prior to any Image Processing being applied to page images, allowing them to be reverted to their original state.
Flag Messages
Type: Embedded Lexicon
A list containing pre-defined flag messages. (See Embedded Lexicon)
