Grooper 2.70 Documentation

Command Options

Defines options for an Object Command in Attended Client.

Inherits from: Connected Object

Properties

Property Name Description
General
Command Type
Type: String
The type of command the options apply to.
Applies To
Type: String
The type of object the command applies to.
Scope
Type: String
The activity or control in which the command is relevant. A scope of 'Global' indicates that the command will appear in all controls and during all activities.
