Grooper 2.70 Documentation

Folder TIF Generator

Defines settings which control the creation of a PDF file from the content associated with a Batch Folder.

Inherits from: Batch Object Stream Generator

Properties

Property Name Description
TIF Options
PDF Render Resolution
Type: Int32, Default: 200
Indicates the resolution at which images will be rendered from PDF pages. This resolution will only be used when rendering PDF pages containing text (PDF pages that are not image-only).
General
Prefer Child Versions
Type: Boolean, Default: False
If true, the output file generation process will prefer content stored on child objects over content stored on parent objects. By default, if a folder contains a PDF or image-based version, then the folder's content will be included in the output, and any content stored on child objects will not be included. Enabling this option allows the output file to be instead constructed using the content stored on child folder or page objects. For example, in a case where PDF files are imported, and image processing has been applied to the pages, it may be desirable to output the processed page versions rather than the original version stored on the folder.
Page Skip Count
Type: Int32, Default: 0
Indicates the number of pages to be skipped during generation of the output file. Defines the number of pages, starting with the first page, which will be excluded from the output. For example, this option could be used to remove covers sheets from the generated output.
