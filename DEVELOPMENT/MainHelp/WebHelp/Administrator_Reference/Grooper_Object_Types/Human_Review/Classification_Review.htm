Grooper 2.70 Documentation

Classification Review
Provides a user interface for reviewing the structure of a batch after [Classification] and/or [Separation]. The user interface for the Classification Review activity includes the [Batch Viewer] Batch View control.

Inherits from: [Attended Client]

Properties

Property Name Description
General
Classification Viewer Settings
Type: [Classification Viewer Settings]
Settings for this control. This optional class allows [Attended Client] activities to expose these settings at the [Batch Process Step] during configuration in Administration. The settings class must inherit from BatchView.ControlSettings. (See [Classification Viewer Settings])
Allow Flagged Folders
Type: [Boolean], Default: False
If enabled, the batch can be completed with flagged folders.
Allow Unclassified Folders
Type: [Boolean], Default: False
If true, the user will be allowed to complete the task while unclassified folders exist. If false, Classify Review will require that all folders at level immediately below the task be classified with a Content Type.
UI Configuration
Command Options
Defines the enabled state and shortcut keys for individual commands in Attended Client. By default, all commands are enabled. Use this property to hide commands which don't apply to the task being performed.
User Activity Timeout
Type: [Int32], Default: 0, Range: 0 - 480
Specifies the amount of time, in minutes, that a user can be inactive before an automatic logout occurs. This is intended to maintain user statistic integrity that could be damaged by users leaving the application open and unattended. A value of 0 will disable this feature.
