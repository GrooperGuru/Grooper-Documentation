ABBYY FineReader

ABBYY FineReader is a commercial OCR Engine recognized as a leader in the industry, and represents a premium OCR engine offering within Grooper.
  • Used for Full Text OCR of machine print documents or Zonal OCR of machine print and/or handwriting.
  • Performs intensive document structure analysis to determine text flow, making it the best option for generating searchable PDF output.
  • Requires separate installation of ABBY FineReader engine.
  • Requires purchase of add-on FineReader volume licensing.
  • Supports 190 languages.


Inherits from: OCR Engine

Properties

Property Name Description
Preprocessing
Correct Inverted Image
Type: Boolean, Default: False
This property set to TRUE tells ABBYY FineReader Engine to detect whether the image is inverted (white text against black background). The text color is detected during page preprocessing, and if it differs from normal, ABBYY FineReader Engine automatically inverts the image.
Correct Shadows && Highlights
Type: ThreeStateValue, Default: Yes
If this property is set to Yes, the image preprocessing will include correction of excessive shadows and highlighting to improve recognition quality. This property is designed for use with photographs only. Can be one of the following values:
  • Yes
  • No
  • Auto
Geometry Correction Mode
Type: CorrectionMode, Default: Auto
Specifies whether geometrical distortions (perspective on photos, curved lines from scanned books, etc.) should be removed during page preprocessing. Can be one of the following values:
  • Auto
  • Enabled
  • Disabled
Resolution Correction Mode
Type: CorrectionMode, Default: Auto
Specifies whether resolution of the image should be corrected during page preprocessing. This property is set to Auto by default, which means that if resolution of the image is suspicious, FineReader Engine should automatically detect image resolution and change it. Can be one of the following values:
  • Auto
  • Enabled
  • Disabled
Orientation Detection
Type: Orientation Detection Params, Default: False
Settings which control orientation detection during OCR.
Skew Correction
Type: Skew Correction Params, Default: Auto
Skew correciton settings.
Analysis
Perform Analysis
Type: Boolean, Default: False
If this property is TRUE, the text areas are detected during layout analysis.
Detect Text
Type: Boolean, Default: False
If this property is TRUE, the text areas are detected during layout analysis.
Table Detection
Type: Table Detection Params, Default: True
Table detection parameters.
Detect Barcodes
Type: Boolean, Default: False
Specifies if barcodes are detected, and accordingly barcode blocks created, during layout analysis. If this property is FALSE, barcodes may be detected as blocks of some other type (e.g. pictures).
Detect Separators
Type: Boolean, Default: False
If this property is TRUE, the separators are detected during layout analysis.
Detect Pictures
Type: Boolean, Default: False
If this property is TRUE, pictures are detected during layout analysis.
Detect Vector Graphics
Type: Boolean, Default: False
If this property is TRUE, vector pictures are detected during layout analysis. Vector picture blocks can appear in the layout only if this property has been set to TRUE during layout analysis.
Recognition
OCR Language
Type: String
The language in which OCR characters should be interpreted.
Allowed Characters
Type: String
The set of allowed characters to be used during OCR.
Language Detection Mode
Type: ThreeStateValue, Default: Yes
Manages automatic language detection. When language autodetection is on, the recognition language is detected for each word in the text. It is selected from the list of languages specified in the Language property. Autodetection is intended to be used during recognition of documents the language of which is not known to you. If you know for certain that all the languages you specified are present in the document, autodetection is useless. Turn it off by setting this property to No. Can be one of the following values:
  • Yes
  • No
  • Auto
Text Types
Type: TextType, Default: 0
The types of text to be recognized. A combination of the following flags:
  • Normal
  • Typewriter
  • Matrix
  • Index
  • Handprint
  • OCR_A
  • OCR_B
  • MICR_E13B
  • MICR_CMC7
  • Gothic
  • Receipt
Recognition Speed
Settings controlling the trade-off between speed and accuracy.
Fine Tuning
Fine tuning settings.
Handprint Settings
Handprint recognition settings.
Synthesis - Page Level
Paragraph Extraction Mode
Type: ParagraphDetectionMode, Default: Normal
Specifies the mode for paragraph extraction. Can be one of the following values:
  • Normal
  • Rough
  • SingleLineParagraphsSpaceFormatting
  • SingleLineParagraphsWordSeparation
Detect Document Links
Type: Boolean, Default: False
If this property is set to TRUE, document references (e.g. cross-references) are detected during page synthesis.
Detect Hyperlinks
Type: Boolean, Default: False
If this property is set to TRUE, hyperlinks are detected during page synthesis.
Synthesize Business Cards
Type: Boolean, Default: False
If this property is set to TRUE, FineReader Engine tries to find business cards on the page and fills in the collections of business cards.
Detect Background Color
Type: ThreeStateValue, Default: Yes
If this property is set to Yes, the background color is detected during page synthesis. Can be one of the following values:
  • Yes
  • No
  • Auto
Allow Gray Background Color
Type: ThreeStateValue, Default: Yes
If this property is set to Yes, the gray color is detected for background. Otherwise, background will be detected as black or white. The value of this property is taken into account only if the DetectBackgroundColor property is set to Yes or Auto. Can be one of the following values:
  • Yes
  • No
  • Auto
Detect Text Color
Type: ThreeStateValue, Default: Yes
If this property is set to Yes, the gray color is detected for background. Otherwise, background will be detected as black or white. Can be one of the following values:
  • Yes
  • No
  • Auto
Allow Gray Background Color
Type: ThreeStateValue, Default: Yes
If this property is set to Yes, the text color is detected during page synthesis. Can be one of the following values:
  • Yes
  • No
  • Auto
Correct Dynamic Range
Type: ThreeStateValue, Default: Yes
If this property is set to Yes, image colors will be corrected so that the background is white and the text is black, or vice versa, which improves image quality. Recognition, however, will slow down. Can be one of the following values:
  • Yes
  • No
  • Auto
