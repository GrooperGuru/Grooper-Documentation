Grooper 2.70 Documentation

Folder Content Generator

Defines settings which control the export of content from a Batch Folder.

Inherits from: Batch Object Stream Generator

Properties

Property Name Description
General
Content Format
Type: TargetContentFormat, Default: Native
The output file format. Can be one of the following values:
  • Native: If a native file version exists, it will be exported. Otherwise, if a PDF version exists, it will be exported. If neither exists, a multipage TIFF file will be created from the child page objects.
  • PDF: Exports a PDF version based on configured PDF export settings.
  • TIF: If the folder has child pages, a multipage TIFF file will be created from the page objects. If no pages exist, but the folder has a PDF version, a multipage TIFF will be created from the PDF version.
PDF Options
Type: Folder PDF Generator
Defines settings which control how PDF content is generated.
TIFF Options
Type: Folder TIF Generator
Defines settings which control how TIFF content is generated.
