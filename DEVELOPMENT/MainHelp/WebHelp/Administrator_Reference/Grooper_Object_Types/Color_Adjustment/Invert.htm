Grooper 2.70 Documentation

Invert
Inverts images which are negated. A "negated" image is one where the feature pixels are white, and the image background is black, as shown below at the left. The output image from this command will be an inverted version of the original image, as shown at the right.



Inherits from: IP Command

Properties

Property Name Description
General
Threshold
Type: Double, Default: 70%, Range: 0% - 100%
The threshold of "darkness" in an image which will trigger inversion. A value of 0% will invert every image. A value of 100% will only invert images which are solid black.
Binarization
Type: Binarize
If enabled, defines binarization settings which will normalize the pixel values in the image to black and white before analysis is performed. (See Binarize)
Command Info
Command Type
Type: String
The type of IP command.
Supported Pixel Formats
Type: String
The pixel formats supported by this command.
