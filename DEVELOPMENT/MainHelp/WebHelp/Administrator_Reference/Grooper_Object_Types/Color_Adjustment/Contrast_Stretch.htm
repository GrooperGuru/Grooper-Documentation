Grooper 2.70 Documentation

Contrast Stretch
Adjusts the contrast of an image so that the lightest pixels are 100% white, and darkest pixels are 100% black. Normalizes the contrast in an image, as shown the in example below.



Inherits from: IP Command

Properties

Property Name Description
General
Black Threshold
Type: String, Default: 5pt
The minimum amount of black area required to establish the black point.
Black Limit
Type: Double, Default: 20%, Range: 0% - 100%
The maximum percentage by which the black point will be adjusted.
White Limit
Type: Double, Default: 20%, Range: 0% - 100%
The maximum percentage by which the white point will be adjusted.
Command Info
Command Type
Type: String
The type of IP command.
Supported Pixel Formats
Type: String
The pixel formats supported by this command.
