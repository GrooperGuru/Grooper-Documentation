Grooper 2.70 Documentation

Colorize
This command can be used to replace the hue, saturation and luminance values in an image with those of the specified base color.

Inherits from: IP Command

Properties

Property Name Description
General
Reference Color
Type: Color, Default: White
The hue, saturation and luminance of the selected color will be used to process the image.
Apply Luminance
Type: Boolean, Default: False
If true, the luminance of the Reference Color will be applied to all pixels in the image.
Apply Saturation
Type: Boolean, Default: True
If true, the saturation of the Reference Color will be applied to all pixels in the image.
Apply Hue
Type: Boolean, Default: True
If true, the hue of the Reference Color will be applied to all pixels in the image.
Command Info
Command Type
Type: String
The type of IP command.
Supported Pixel Formats
Type: String
The pixel formats supported by this command.
