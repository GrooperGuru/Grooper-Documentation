Grooper 2.70 Documentation

Grooper Root
Represents the root node of a Grooper repository. All [Grooper Object Types] in a Grooper repository are stored as children or descendants of the root node. The root node also stores several settings which apply to the repository.

Inherits from: [Grooper Node]

JPEG Quality
Type: [Double], Default: 75%, Range: 1% - 100%
The JPEG compression quality to use for internal storage of color images in Grooper. Valid values are 1 - 100% with a default of 75%. The lower the quality, the smaller the filesize. A good range of values is between 70% and 95%.
JPEG Smoothing
Type: [Double], Default: 0%, Range: 0% - 100%
The smoothing level to apply when saving color images, prior to JPEG compression. Smoothing improves compression, sometimes at the expense of image quality. Valid values are 0% to 100% with a default of 0%. Increasing the smoothing value will tend to create smaller Jpeg files, however the image is blurred while saving.
Enabled Logging Events
Type: [EventType], Default: Err, Wrn, Prc, Inf, Aud
Event types to be logged to the system log table. A combination of the following flags:
  • Err: Errors - Critical error messages.
  • Wrn: Warnings - Non-critical warning messages.
  • Prc: Procedural - Tracing of procedural calls. NOTE: Enabling this option can generate large volumes of log entries. Use only during troubleshooting.
  • Inf: Information - Detailed information logging. NOTE: Enabling this option can generate large volumes of log entries. Use only during troubleshooting.
  • Aud: Audit - Audit trail events such as batch creation, object deletion, etc.
