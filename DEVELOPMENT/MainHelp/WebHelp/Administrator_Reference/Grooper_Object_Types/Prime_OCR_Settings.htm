Grooper 2.70 Documentation

Prime OCR Settings



Inherits from: Ocr Engine Settings

Properties

Property Name Description
Recognition
OCR Level
Type: OcrLevel, Default: 0
The level of OCR to perform. Can be one of the following values:
  • Level1: Standard OCR accuracy and performance.
  • Level2: Compared to Level1: 15% better character accuracy and much better suspicious error marking. 2.1X slower.
  • Level3: Compared to Level1: 58% better character accuracy and much better suspicious error marking. 3.1X slower
  • Level4: Compared to Level1: 74% better character accuracy and much better suspicious error marking. 5.8X slower.
  • Level5: Compared to Level1: 76% better character accuracy and much better suspicious error marking. 7.1X slower.
  • Level6: Compared to Level1: 78% better character accuracy and much better suspicious error marking. 8.0X slower.
Allowed Character Set
Type: ZoneRestrictionType, Default: Any
The set of allowed characters. Can be one of the following values:
  • Any: Any character type accepted.
  • Alpha: Only alpha characters. (A-Za-z)
  • Numeric: Only numeric characters. (0-9,.+-()$/@#)
  • Any_UpperCase: Any character, but only upper case alpha characters.
  • Any_LowerCase: Any character, but only lower case alpha characters.
  • Alpha_UpperCase: Only upper case alpha characters. (A-Z)
  • Alpha_LowerCase: Only lower case alpha characters. (a-z)
Use Lexicon
Type: Boolean, Default: False
Indicates whether Prime OCR should use a lexicon to validate and correct OCR results.
