Grooper 2.70 Documentation

Color Detection
Detects whether an image contains color, grayscale, or black and white content, optionally converting the image to the detected color depth.

Inherits from: IP Command

Properties

Property Name Description
General
Minimum Saturation
Type: Double, Default: 10%, Range: 0% - 100%
Defines the color sensitivity. Lower values will detect fainter colors. Review the Candidates intermediate image after adjusting this property. The value entered here is used to threshold the Saturation image, and should be tuned so that color features are clearly visible on the Candidates image.
Minimum Distance From Gray
Type: Double, Default: 6%, Range: 0% - 100%
Defines the minimum distance from pure gray required for a pixel to be interpreted as color. Review the "Preprocessed" intermediate image after adjusting this property. The value should be tuned so that the gray features on the image are largely removed, and all color areas are intact.
Minimum Color Region Size
Type: Unit Size, Default: 4pt
The minimum size of a color region which will trigger a positive result. Specifies the minimum size of a blob on the Candidates image which will trigger a positive result.
Processing Resolution
Type: ProcessingResolution, Default: Dpi100
The image resolution at which color detection will be performed. Can be one of the following values:
  • FullResolution
  • Dpi10
  • Dpi15
  • Dpi20
  • Dpi25
  • Dpi30
  • Dpi40
  • Dpi50
  • Dpi75
  • Dpi100
  • Dpi150
  • Dpi200
  • Dpi250
  • Dpi300
Represents a trade-off between speed and accuracy during color detection. Lower resolutions increase speed, at the risk of reduced accuracy.
