Sticky Note Detection
Detects color sticky notes, optionally flagging the pages on which they are detected. This command is useful in helping scan operators identify pages with sticky notes attached at scan time, so that any corrective action can be taken before the batch is completed.

This command is designed to detect color sticky notes on non-color document backgrounds. Sticky note detection will not work on documents which have a colored background.

Minimum Gray Distance
Type: Double, Default: 10%, Range: 1% - 100%
The percentage deviation from pure gray which represents "color". Defines the overall color sensitivity. Lower values will detect fainter colors. Review the Dropout image after making changes to this property. The objective when tuning this property is to produce a Dropout image containing only color elements, and a Candidates image where each sticky note produces a solid white rectangle.
Corner Tolerance
Type: String, Default: 0.75in
For the purposes of corner detection, specifies the maximum distance between the ends of two perpendicular lines. After performing edge detection, the algorithms performs line detect and searches for lines which form a rectangle. This setting controls how much tolerance there is for lines which don't necessarily touch each other.
Angle Tolerance
Type: Double, Default: 2°
The maximum angle variance between line segments forming a rectangle. When searching for rectangles, the algorithm is looking for 4 lines, where 2 of the lines are perpendicular to the other 2. The definition of perpendicular is that the line angles differ by +- 90°. Due to the imperfect nature of scanned images, the actual angles will vary based on resolution and quality. This setting controls how much variation is allowed.

Flag Pages
Type: Boolean, Default: True
If set true, pages on which one or more sticky notes are detected will be flagged.
