Grooper 2.70 Documentation

Edge Detection
Detects edges in an image using a Sobel filter.

Inherits from: IP Command

Properties

Property Name Description
General
Low Threshold
Type: Double, Default: 8
Low threshold value used for edge detection.
High Threshold
Type: Double, Default: 12
High threshold value used for edge detection.
Dilation Factor
Type: Int32, Default: 0
The factor by which edges will be dilated. A value of 0 will produce edges 1 pixel wide. A non-zero value N will produce edges N x 2 + 1 pixels wide.
Invert
Type: Boolean, Default: False
Determines whether edges are detected inside or outside of objects.

If set to false, edges will be detected around the outside of objects. If set to true, edges will be detected around the inside of objects.
