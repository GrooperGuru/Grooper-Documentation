Grooper 2.70 Documentation

CMIS Connection
A CMIS Connection object represents a connection to CMIS-compliant content server. For a detailed overview of CMIS, please refer to the [Content Management System Connectivity] topic.

Inherits from: [Grooper Node]

Properties

Property Name Description
General
URL
Type: [String]
The CMIS AtomPub URL
Authentication Mode
Type: [AuthenticationProvider], Default: Basic
The user account to be used for accessing the [CMIS repository]. Can be one of the following values:
  • Basic: Uses basic authentication.
  • NTLM: Uses NTLM authentication.
User Name
Type: [String]
The user account to be used for accessing the [CMIS repository].
Description
Type: [String]
Generic property allowing an administrator to document the purpose of this Grooper Node.
Connection Settings
Connection Timeout
Type: [Int32], Default: 0
The timeout, in seconds, to be used for connections.
Read Timeout
Type: [Int32], Default: 0
The timeout, in seconds, to be used when reading data from the server.
Query Timeout
Type: [Int32], Default: 0
The timeout, in seconds, to be used when running queries.
Content Stream Timeout
Type: [Int32], Default: 0
The timeout, in seconds, to be used when reading or writing the content stream for a document.
Use Compression
Type: [Boolean], Default: True
Controls whether HTTP compression is used during data transfers.
