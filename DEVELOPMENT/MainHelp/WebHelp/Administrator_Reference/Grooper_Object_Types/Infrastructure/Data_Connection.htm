Grooper 2.70 Documentation

Data Connection
A Data Connection object represents a connection to a database, storing all information necessary to access the database. These objects provide a connection basis for [Database Table] objects, and are also used directly in scripts to avoid storing authentication credentials in script code.

Inherits from: [Grooper Node]

Properties

Property Name Description
General
Provider
Type: [DatabaseProvider], Default: SqlServer
The database provider to be used for this connection. Can be one of the following values:
  • SqlServer: Microsoft SQL Server
  • ODBC: Any ODBC-compliant database provider.
Connection Settings
Connection settings for the selected provider.
Description
Type: [String]
Generic property allowing an administrator to document the purpose of this Grooper Node.
