Row Detector

The row detection process uses the Value Extractors defined on primary columns to detect individual table rows.

The general operation of row detection is as follows:

  • The value extractors for each primary column are executed against the source content.
  • Results occuring above the header row or below the footer row are discarded.
  • Text lines containing a least one result for each primary column are identified as "primary lines".
  • If the primary line contains only one possible value for a primary column, the value is accepted without checking the header.
  • If the primary line contains multiple possible values for a primary column, header information is used to resolve the conflict.
  • Each primary line is then adjusted to consume additional text lines associated with the table row.


Inherits from: Grooper Object

Properties

Property Name Description
General
Max Row Height
Type: String, Default: 2in
The maximum geometrical height of any row in the table. If this value is empty or zero, no height limit will be enforced. This means that the first row in the table may extend all the way up to the header, the last row can extend all the way to the footer, and middle rows will consume all content between row boundaries.

If a maximum height is specified, this indicates the maximum vertical distance to be consumed by any row. This limits how far the first row may extend upwards toward the header, how far the last row may extend downwards toward the footer, and how much total space middle rows may consume.
Max Vertical Gap
Type: String, Default: 0.5in
For the first and last rows of a table, defines the maximum amount of white space between lines of text which should be included in the row content.
Indent Size
Type: String, Default: 0.2in
The amount of horizontal space to be considered indentation during row analysis. If this value is empty or zero, indentation will not be considered when determining row layout.
Max Font Size Variation
Type: Double, Default: 20%, Range: 0% - 100%
The maximum percentage variation in font size between text lines which are part of the same row.
Max Row Lines
Type: Int32, Default: 0
The maximum number of text lines which a single table row may consume. A value of 0 indicates no limit.
