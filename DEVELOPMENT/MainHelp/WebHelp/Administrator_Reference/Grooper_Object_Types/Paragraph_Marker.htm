Paragraph Marker

Defines settings which control paragraph detection. Paragraph detection is a text preprocessing option which modifies the input so that \r\n pairs appear only at the end of paragraphs.

Inherits from: Embedded Object

Properties

Property Name Description
General
Maximum Line Spacing
Type: Double, Default: 0.4
The maximum height, in inches, of the line spacing inside a paragraph. Any vertical line gap larger than this value will be considered the start of a new paragraph.
Paragraph Spacing Ratio
Type: Double, Default: 125%, Range: 0% - 500%
The percentage increase in line spacing which should trigger a new paragraph. This setting only impacts paragraphs which contain at least two lines, allowing an average line spacing to be calculated. If a vertical gap is encountered which is larger than the average line spacing times the Paragraph Spacing Ratio, it will be considered the start of a new paragraph.
Indent Size
Type: Double, Default: 0.2
The minimum width, in inches, of an indentation which should be considered the start of a new paragraph. Specified in inches. A value of 0 will disable automatic insertion of vertical tabs based on indentation.
Detect Bullets
Type: Boolean, Default: True
If true, bulleted paragraph headings will be detected. Note that custom bullet detection can be implemented using the First Line Extractor property.
Detect Double Space
Type: Boolean, Default: True
If true, paragraphs with double spacing will be formatted with double CR/LF pairs.
First Line Extractor
Type: Embedded Extractor
An optional extractor to identify the first line of a paragraph. Vertical tabs will be inserted at the beginning of every line where this extractor produces a match. (See Embedded Extractor)
Preprocessing Options
Minimum Line Width
Type: Double, Default: 3
Specifies the minimum width, in inches, of a text line for which should flow to the following line.
Maximum Horizontal Gap
Type: Double, Default: 0.5
Specifies the maximum horizonal gap, in inches, which can appear in a line which will flow to the following line.
