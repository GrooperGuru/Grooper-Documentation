Embedded Lexicon

Represents a property which can include locally-defined lexicon entries as well as references to external lexicons.

Inherits from: Embedded Object

Properties

Property Name Description
General
Type
Type: LexiconType, Default: Simple
Specifies how entries in the lexicon will be interpreted. Can be one of the following values:
  • Simple: A simple lexicon contains a list of key values, one value per line.

    about
    above
    after
    again
    against
    ...

  • Lookup: A lookup lexicon contains key-value pairs delimited by '='.

    AL=Alabama
    AK=Alaska
    AZ=Arizona
    AR=Arkansas
    CA=California
    ...

  • Vocabulary: A vocabulary lexicon contains key-frequency pairs delimited by '='.

    you=22484400
    i=19975318
    the=17594291
    to=13200962
    a=11230036
    ...
Type
Type: LexiconType, Default: Simple
 
Entries
Type: String
A list of local lexicon entries. The list should be formatted so that there is one entry per line. Use the = symbol to indicate replacement values.
Included Lexicons
One or more lexicons to include. Only lexicon
