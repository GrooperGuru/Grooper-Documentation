Grooper 2.70 Documentation

Image Info



Inherits from: Grooper Object

Properties

Property Name Description
General
Pixel Format
Type: PixelFormat, Default: 0
The pixel format of the primary image. Can be one of the following values:
  • Pixel8bppGrayscale: 8-Bit Grayscale
  • Pixel24bppBgr: 24-Bit Color
  • Pixel32bppBgr: 32-Bit Color
  • Pixel1bppIndexed: Black and White
  • Pixel4bppIndexed: 4-Bit Paletted
  • Pixel8bppIndexed: 8-Bit Paletted
  • Pixel16bppGrayscaleAlpha: 16-Bit Grayscale Alpha
  • Pixel16bppGrayscale: 16-Bit Grayscale
  • Pixel48bppBgr: 48-Bit Color
  • Pixel32bppBgra: 32-Bit Color Alpha
  • Pixel64bppBgra: 64-Bit Color Alpha
  • Pixel32bppCmyk: 32-Bit CMYK
Size
Type: String
The width and height of the image, in pixels.
Logical Size
Type: String
The width and height of the image, in inches.
Resolution
Type: String
The horizontal resolution of the primary image in DPI.
Bit Depth
Type: Int32, Default: 0
The bit depth of the primary image.
Compressed File Size
The compressed file size of the primary image.
Uncompressed Image Size
The uncompressed raw image size of the primary image.
Compression Ratio
Type: Double, Default: 0%
The compression ratio.
