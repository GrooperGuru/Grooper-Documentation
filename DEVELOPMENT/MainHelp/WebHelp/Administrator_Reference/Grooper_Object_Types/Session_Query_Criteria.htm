Session Query Criteria

Defines search criteria for session statistics.

Inherits from: Connected Object

Properties

Property Name Description
General
Date Range
Type: TimeRange, Default: All
Defines the date range for the activity start time. Can be one of the following values:
  • All: Includes all sessions, regardless of start time.
  • Today: Includes sessions started today.
  • Yesterday: Includes sessions started yesterday.
  • MTD: Includes sessions started between the beginning of the current month and today.
  • LastMonth: Includes sessions started last month.
  • YTD: Includes sessions started between the beginning of the current year and today.
  • LastYear: Includes sessions started last year.
  • Custom: Specify a specific start and/or end date.
Start Date
When the date range is 'Custom', defines the starting date for included entries.
End Date
When the date range is 'Custom', defines the starting date for included entries.
