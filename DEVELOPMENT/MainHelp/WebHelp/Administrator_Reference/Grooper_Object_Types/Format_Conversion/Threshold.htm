Grooper 2.70 Documentation

Threshold
Converts an image to black and white using a variety of thresholding methods.

Inherits from: Binarize

Properties

Property Name Description
General
Thresholding Method
Type: BinarizationMethod, Default: Auto
Specifies the thresholding method which will be used. Can be one of the following values:
  • Simple: Thresholds an image to black and white using a fixed threshold value between 0 and 255.
  • Auto: Selects a threshold value automatically based on histogram analysis.
  • Adaptive: Thesholds pixels based on the intensity of pixels in the local neighborhood.
  • Dynamic: Performs adaptive thresholding, while preserving dark areas on the page.
Black Grouping
Type: Double, Default: 25%, Range: 0% - 100%
In dymamic thresholding, specifies how aggressively dark areas of the image will be preserved.
Threshold
Type: Int32, Default: 127, Range: 1 - 255
Defines the threshold value to be used for simple thresholding.
Window Size
Type: String, Default: 5pt
The size of the window used to determine the threshold value for each pixel in Dynamic and Adaptive thresholding.
Kernel Type
Type: AdaptiveKernelType, Default: Box
Specifies the kernel type to use for adaptive thresholding. Can be one of the following values:
  • Box: Uses a box kernel.
  • Gaussian: Uses a Gaussian kernel.
Delta
Type: Single, Default: 16, Range: 1 - 255
For dynamic and adaptive thresholding, specifies the change in intensity required for a pixel to be considered an edge pixel.
Sigma
Type: Single, Default: 0, Range: 0 - 255
For adaptive thresholding using a Gaussian kernel, specifies the Gaussian sigma value. If set to zero, the sigma value will be calculated automatically based on the Mask Window Size, using the following formula: sigma = 0.3 * ((MaskWindowSize - 1) * 0.5 - 1) + 0.8.
Command Info
Command Type
Type: String
The type of IP command.
Supported Pixel Formats
Type: String
The pixel formats supported by this command.
