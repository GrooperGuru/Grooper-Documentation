Grooper 2.70 Documentation

Scanner Profile
Defines a connection to a capture device, and settings which control how the device operates. Scanner Profiles are typically created and edited using the [Scan Control] - a component of the [Batch Scan Control] used by the [Scan] activity. To create, edit, and test Scanner Profiles from Grooper Administration, go to any [Batch Process Step] which is configured for the [Scan] activity, and select the [Attended Activity Tester] tab.

Inherits from: [Grooper Node]

Generic property allowing an administrator to document the purpose of this Grooper Node.
Options
Feed Orientation
Type: [FeedOrientation], Default: TopFirst
Indicates the orientation in which pages will be scanned. Can be one of the following values:
  • TopFirst: No rotation will be performed.
  • LeftFirst: Front pages will be rotated 90° counter-clockwise, and back pages will be rotated 90° clockwise.
  • RightFirst: Front pages will be rotated 90° clockwise, and back pages will be rotated 90° counter-clockwise.
  • BottomFirst: All pages will be rotated 180°.
  • Auto: Grooper will automatically determine the orientation of each page and rotate it accordingly.
Scan Area
Type: [Rectangle], Default: (0,0):(0,0)
The rectangular scan area to configure the scanner with, specified in inches. If this property is set to any value other than a blank rectangle (0,0):(0,0), then this rectangle will be used as the capture area during scan. This will override the scan area which would otherwise be calculated from the scanner driver's page size and cropping settings.
IP Profile
Type: [IP Profile]
An optional [IP Profile] to be used for scanner-specific image cleanup. In some cases, it may be necessary to apply custom image cleanup to the images produced by a specific scanner model. In such cases, an IP Profile should be created and attached to the Scanner Profile rather than the Scan module settings, so that it only applies to images captured using the scanner model. NOTE: Real time permanent image cleanup should be applied using the IP Profile attached to the Scan module. (See [IP Profile])
ScanOnce™
ScanOnce™ Mode
Type: [SimplePixelFormat], Default: None
The ScanOnce™ operating mode. Controls whether ScanOnce™ is enabled, and specifies the default color format in which pages are displayed. Can be one of the following values:
  • None: None.
  • Grayscale: 8-Bit Grayscale
  • Color: 24-Bit Color
  • Binary: Black and White
To enable ScanOnce™, configure the scanner to scan in color or grayscale, and then set this property to indicate the default display format for each captured page. To disable ScanOnce™ entirely, set the ScanOnce™ Mode to 'None'.
Detect Color
Type: [Boolean], Default: False
Enables or disables automatic color detection. When automatic color detection is enabled, Grooper will analyze each page to determine if it contains color. If the page contains color, the display format will be set to color rather than the default specified in the ScanOnce™ Mode property. The sensitivity of color detection can be adjusted using the Color Detection Settings property.
Color Detection Settings
Type: [Color Detection]
The settings used to determine if a page contains color. (See [Color Detection])
Threshold Method
Type: [BinarizationMethod], Default: Simple
The default thresholding method to be used for displaying pages in black and white. Can be one of the following values:
  • Simple: Thresholds an image to black and white using a fixed threshold value between 0 and 255.
  • Auto: Selects a threshold value automatically based on histogram analysis.
  • Adaptive: Thesholds pixels based on the intensity of pixels in the local neighborhood.
  • Dynamic: Performs adaptive thresholding, while preserving dark areas on the page.
Default Threshold
Type: [Int32], Default: 0
The threshold value to be used when Simple or Global thresholding is in use. For Simple thresholding, this value determines the amount of intensity required for a pixel to be considered white. For Global threshold, this value serves as the 'initial threshold' value required for global thresholding.

Threshold values can range from 0 to 255, with 0 representing pure black and 255 representing pure white.
Default Brightness
Type: [Int32], Default: 0
The default brightness adjustment used for pages displayed in color or grayscale. This a value between -100% and 100% representing the percentage by which the brightness will be increased or decreased."
Default Contrast
Type: [Int32], Default: 0
The default contrast adjustment used for pages displayed in color or grayscale. This a value between -100% and 100% representing the percentage by which the contrast will be increased or decreased."
