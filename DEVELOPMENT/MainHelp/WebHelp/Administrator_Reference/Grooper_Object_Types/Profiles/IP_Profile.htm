IP Profile
An IP Profile defines a sequence of image processing operations which are to be applied to an image. IP Profiles can have [IP Step] and [IP Group] objects as children. A simple IP Profile might be a sequence of [IP Step] objects, each specifying an [IP Command] to be executed. A more complex IP Profile might be a hierarchy of groups and steps, with conditional execution logic to customize flow based on properties of the image.

The execution order of items in an IP Profile is based on a pre-order tree traversal, which simply means that items execute in the order in which they would appear if the entire tree structure were expanded. The execution order can be customized by specifying flow control expressions on individual [IP Group] or [IP Step] objects



Inherits from: [Ip Element Container]

Properties

Property Name Description
General
Should Execute Expression
Type: [String]
Defines an expression which determines whether the [IP Step] or [IP Group] should execute. Must evaluate to Boolean value. True means the item should execute, and false means it should be skipped. Below are various examples of Should Execute expressions:

Execute only if the input image is color:
Image.IsColor

Execute only if the input image is black and white:
Image.IsBinary

Execute only if the input image is 8-bit grayscale:
Image.ColorFormat=GrooperImage.PixelFormat.Pixel8bppGrayscale

Execute only if the previously-executed [Measure Entropy] command returned an entropy of more than 85%:
Results.Measure_Entropy.Entropy > 0.85#

Execute only if the previously-executed [Classify Image] command returned a Class Name of "Noisy Blue Background":
Results.Classify_Image.ClassName = "Noisy Blue Background"
Next Step Expression
Type: [String]
Defines an expression which determines the next IP Step to execute. Fires after a [Batch] completes the current step, and must evaluate to an [IP Step] or [IP Group] object. If Nothing is returned, execution of the [IP Profile] will terminate. Otherwise, execution will resume at the step returned. Below are various examples of Next Step expressions:

If the image is black and white, run the 'Binary Cleanup' [IP Group]. Otherwise, run the 'Color Cleanup' group:
If(Image.IsBinary, Binary_Cleanuup, Color_Cleanup)

If the image is black and white, terminate execution. Otherwise, proceed to the 'Color Detect' [IP Step]:
If(Image.IsBinary, Nothing, Color_Detect)
Description
Type: [String]
Generic property allowing an administrator to document the purpose of this Grooper Node.
