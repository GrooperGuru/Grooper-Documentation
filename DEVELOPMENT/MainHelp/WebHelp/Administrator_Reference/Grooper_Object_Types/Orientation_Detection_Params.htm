Grooper 2.70 Documentation

Orientation Detection Params



Inherits from: Grooper Object

Properties

Property Name Description
General
Correct Orientation
Type: Boolean, Default: False
If this property is TRUE, the page orientation is detected during page preprocessing, and if it differs from normal, ABBYY FineReader Engine automatically rotates the image. Note - If this property is set to TRUE, then the TextTypes property under 'Recognition' cannot be set to 'Handwriting'
Detect Orientation Mode
Type: OrientationDetectionMode, Default: Fast
Specifies the mode of page orientation detection. Can be one of the following values:
  • Fast
  • Normal
  • Thorough
Prohibit Clockwise Rotation
Type: Boolean, Default: False
Disables clockwise page rotation when selecting the page orientation.
Prohibit Counter Clockwise Rotation
Type: Boolean, Default: False
Disables counterclockwise page rotation when selecting the page orientation.
Prohibit Upsidedown Rotation
Type: Boolean, Default: False
Disables upside-down page rotation when selecting the page orientation.
