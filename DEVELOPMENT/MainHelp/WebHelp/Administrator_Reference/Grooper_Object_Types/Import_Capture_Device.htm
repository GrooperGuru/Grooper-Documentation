Import Capture Device

Implements a virtual scanner by importing from a file system directory. This scanner device does not use an actual document scanner. Instead, it is configured with a file system path. Each time a scan is initiated, the contents of the folder are imported as a linear stream of images, simulating the stream of images which come from a document scanner.

Inherits from: [Direct Capture Device]

Properties

Property Name Description
Import Settings
Import Source
Type: [ImportType], Default: FileSystem
The source from which files will be imported. Can be one of the following values:
  • FileSystem: Simulate a scanner by importing files from a directory.
  • TestBatch: Simulate a scanner by importing files from a test [batch].
Source Batch
Type: [Batch]
The [batch] from which files will be imported. (See [Batch])
Import Path
Type: [String]
The path from which files will be imported.
Delete Files
Type: [Boolean], Default: False
If true, files will be deleted upon successful import.
Color Format Conversion
Type: [SimplePixelFormat], Default: None
The color format to convert each page to. If set to None, pages will be imported in the color format of the source file. Can be one of the following values:
  • None: None.
  • Grayscale: 8-Bit Grayscale
  • Color: 24-Bit Color
  • Binary: Black and White
Scanning Speed
Type: [ScanningSpeed], Default: Speed_Unlimited
Throttles the import speed to simulate the speed of a document scanner. Can be one of the following values:
  • Speed_Unlimited: Speed throttling disabled. Images will be imported as fast as possible.
  • Speed_25ipm: 25 images per minute
  • Speed_50ipm: 50 images per minute
  • Speed_100ipm: 100 images per minute
  • Speed_150ipm: 150 images per minute
  • Speed_200ipm: 200 images per minute
  • Speed_250ipm: 250 images per minute
  • Speed_300ipm: 300 images per minute
