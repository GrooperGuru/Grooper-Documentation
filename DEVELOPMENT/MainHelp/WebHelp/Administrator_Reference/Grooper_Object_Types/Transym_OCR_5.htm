Transym OCR 5

Transym OCR 5 is a commercial OCR engine which provides highly accurate multilanguage language OCR.
  • Used for Full Text and Zonal OCR of documents containing machine print characters.
  • Does not support handwriting.
  • Fully installed with Grooper.
  • Requires 1 license per concurrent process.
  • Supports 28 languages.


Inherits from: OCR Engine

Properties

Property Name Description
Image Cleanup
Auto Invert Zones
Type: Boolean, Default: True
If enabled, the engine will attempt to automatically invert white-on-black zones.
Deskew
Type: Boolean, Default: True
If enabled, the image will be automatically deskewed prior to OCR.
Deshade
Type: Boolean, Default: True
If enabled, a deshade image cleanup process will be executed prior to OCR.
Invert Whole Page
Type: Boolean, Default: False
If enabled, the entire page will be inverted prior to OCR.
Noise Removal
Type: Boolean, Default: True
If enabled, noise removal will be performed prior to OCR.
Line Removal
Type: Boolean, Default: True
If enabled, line removal will be performed prior to OCR.
Document Structure
Orientation
Type: PageOrientation, Default: Auto
Indicates the orientation of text to the extracted. Can be one of the following values:
  • Auto: Transym will automatically detect the page orientation.
  • Angle90: Transym will assume that all pages are rotated 90°.
  • Angle180: Transym will assume that all pages are rotated 180°.
  • Angle270: Transym will assume that all pages are rotated 270°.
  • None: Transym will assume that all pages are upright and require no rotation.
Perform Sectioning
Type: Boolean, Default: False
If enabled, the OCR engine will attempt to detect the structure of the document during OCR processing.
Processing Options
Merge Broken Characters
Type: Boolean, Default: True
If enabled, broken characters will be merged prior to OCR.
Reject Questionable Lines
Type: Boolean, Default: False
If enabled, the OCR engine will attempt to detect entire lines of 'junk' and remove them.
Reject Questionable Characters
Type: Boolean, Default: False
If enabled, individual characters with questionable confidence values will be rejected.
Lexicon Mode
Type: LexMode, Default: Auto
Specifies the lexicon mode. Can be one of the following values:
  • Auto: The OCR engine will automatically determine if lexicon processing should be applied.
  • Enabled: Lexicon processing is enabled.
  • Disabled: Lexicon processing is disabled.
Run Speed
Type: JobSpeed, Default: Slow
Specifies the speed/accuracy trade-off. Can be one of the following values:
  • Slow
  • Medium
  • Fast
  • Express
Language
Allowed Languages
Type: [List `1], Default: English
The set of allowed languages for the OCR process.
Auto Localize
Type: Boolean, Default: False
If true, automatically localizes the output culture to the primary region for the language. For example, will convert 'en' to 'en-US', 'it' to 'it-IT', and etc.
Allowed Characters
The set of allowed characters for the OCR process.
