Grooper 2.70 Documentation

Correct OCR
Corrects OCR results for a document, or for a data element on a document. Performs word splitting and OCR spell correction.

Inherits from: Unattended Activity

Properties

Property Name Description
General
Scope
Type: RepairScope, Default: Document
Indicates the scope of the repair operation. Can be one of the following values:
  • Document: The extractor will be executed against the full document content.
  • DataElement: The extractor will be executed against the content of a Data Element. If word splitting is to be performed on a data element, the data element must have its Preserve Character Data property set to true.
Data Element
Type: Data Element
When Scope is set to 'DataElement', indicates the DataElement to be repaired. If word splitting is to be enabled, the 'Preserve Character Data' property of the data element must be set to true. (See Data Element)
Spell Correction
Enable Spell Correction
Type: Boolean, Default: False
Indicates whether spell correction should be performed. Spell correction works by executing an extractor against the source, and then modifying the source to reflect the return value of the extractor. For example, to spell correct words, simply define an extractor which matches all words, and fuzzy match it against a lexicon of words.
Correction Extractor
Type: Embedded Extractor
An extractor to be used for matching text segments which should be corrected. All instances where this extractor hits will be replaced with the output value of the extractor. (See Embedded Extractor)
Removal Extractor
Type: Embedded Extractor
An extractor to be used for matching text segments which should be removed. All instances where this extractor hits will be replaced with the output value of the extractor. (See Embedded Extractor)
Word Splitting
Enable Word Splitting
Type: Boolean, Default: False
Indicates whether word splitting should be performed. Word splitting helps to correct imperfections in word spacing, which are typically introduced when OCR engines process degraded document images.

Word splitting works by examining each distinct text element in scope. A 'text element' is defined as an unbroken sequence of alpha characters with a length which satifies the Minimum Word Length property. Each text element may optionally contain numeric characters at the beginning or end.

If the value of a text element does not appear in the Word Split Vocabulary then it will be analyzed to see if it is multiple words stuck together. This is done by first separating the sequence at the largest whitespace gaps. For example, assume the term 'meetstatutory' is encountered. Since 'meetstatutory' does not appear in the vocabulary, we split it at the largest gap and get 'meet' and 'statutory'. If both of these terms appear in the vocabulary, the term will be split.

If any of the terms are numeric, they will qualify for splitting so long as the alpha terms appear in the vocabulary.
Minimum Word Length
Type: Int32, Default: 0
Indicates the minimum length of a word which will be analyzed for splitting.
Word Split Vocabulary
Type: Embedded Lexicon
Provides the vocabulary to be used for word splitting. (See Embedded Lexicon)
Fuzzy Match Similarity
Type: Double, Default: 100%, Range: 0% - 100%
The fuzzy match similarity to be used when looking up the individual words during a split analysis. A value of 100% requires an exact match, effectively disabling fuzzy matching.
Max Entries
Type: Int32, Default: 0
 
Error Handling
Error Disposition
Type: IssueDisposition, Default: Flag, Log
Determines what happens when an error occurs processing an activity. A combination of the following flags:
  • None: The issue will be ignored, and the item will complete successfully.
  • Flag: The associated Batch Folder or Batch Page will be flagged.
  • Log: The issue will be logged to the Grooper log.
  • Stop: The Batch will be set to error.
Maximum Consecutive Errors
Type: Int32, Default: 0
The maximum number of consecutive errors, after which a critical stop will be raised. A critical stop will cause services to stop running.
