Grooper 2.70 Documentation

Image Review
Image Review is an attended activity which provides a user interface optimized for quickly reviewing and/or correcting the results of Classification or Separation. The user interface for the Image Review activity includes the Batch Viewer and Thumbnail Viewer Batch View controls.

Inherits from: Attended Client

Properties

Property Name Description
General
Pre-Defined Flag Messages
Type: Lexicon
A list containing pre-defined page flag messages. (See Lexicon)
Allowed IP Profiles
IP Profiles available to operators during image review.
Require Review of All Pages
Type: Boolean, Default: True
When set to True, Batches or documents can not be completed unless all pages are marked as reviewed.
Allow Completion with Flagged Pages
Type: Boolean, Default: True
When set to True, Batches or documents with flagged pages can be completed.
UI Configuration
Command Options
Defines the enabled state and shortcut keys for individual commands in Attended Client. By default, all commands are enabled. Use this property to hide commands which don't apply to the task being performed.
User Activity Timeout
Type: Int32, Default: 0, Range: 0 - 480
Specifies the amount of time, in minutes, that a user can be inactive before an automatic logout occurs. This is intended to maintain user statistic integrity that could be damaged by users leaving the application open and unattended. A value of 0 will disable this feature.
