Grooper 2.70 Documentation

Review
Provides a configurable user interface for human review of documents or pages in a batch. The Attended Client provides the user interface for this Activity. This client will display one or more Batch View controls as indicated by the Activity's configuration.

Inherits from: Attended Client

Properties

Property Name Description
UI Configuration
Batch Views
Defines the collection of Batch Views that will be available for this Activity.
Command Options
Defines the enabled state and shortcut keys for individual commands in Attended Client. By default, all commands are enabled. Use this property to hide commands which don't apply to the task being performed.
User Activity Timeout
Type: Int32, Default: 0, Range: 0 - 480
Specifies the amount of time, in minutes, that a user can be inactive before an automatic logout occurs. This is intended to maintain user statistic integrity that could be damaged by users leaving the application open and unattended. A value of 0 will disable this feature.
Completion Criteria
Allow Completion with Flagged Pages
Type: Boolean, Default: False
Allow the completion of items that contain flagged pages.
All Pages at Level
Type: Int32, Default: 0
Allow the completion of items only when all Batch Pages are at this level. Leave the value set to zero (default) if items can be completed with pages at any level.
Folder Criteria
Completion criteria options for specific Batch Folder levels. Configuring criteria here allows administrators to restrict the user's ability to complete tasks.
