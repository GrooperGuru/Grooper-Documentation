Grooper 2.70 Documentation

Redaction
Applies a Redaction Profile to pages in a batch. Redaction performs permanent image-based redaction of information based on settings defined in a Redaction Profile.

Inherits from: Unattended Activity

Properties

Property Name Description
General
Redaction Color
The color to use for drawing redactions. If no color is specified, the background color of the image will be used.
Transparency
Type: Double, Default: 0%, Range: 0% - 100%

Type: String
The minimum output format, based on the redaction color.
Create Undo Image
Type: Boolean, Default: False
 
Redaction Extractors
Data Types
Data types which will be used to redact information.
Redaction Fields
Fields which will be redacted.
Error Handling
Error Disposition
Type: IssueDisposition, Default: Flag, Log
Determines what happens when an error occurs processing an activity. A combination of the following flags:
  • None: The issue will be ignored, and the item will complete successfully.
  • Flag: The associated Batch Folder or Batch Page will be flagged.
  • Log: The issue will be logged to the Grooper log.
  • Stop: The Batch will be set to error.
Maximum Consecutive Errors
Type: Int32, Default: 0
The maximum number of consecutive errors, after which a critical stop will be raised. A critical stop will cause services to stop running.
