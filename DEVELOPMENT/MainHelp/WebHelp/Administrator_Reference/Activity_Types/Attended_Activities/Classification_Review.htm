Classification Review

Provides a user interface for reviewing the structure of a batch after Classification and/or Separation. The user interface for the Classification Review activity includes the Batch Viewer Batch View control.

Properties

Property Name Description
General
Classification Viewer Settings Settings for this control. This optional class allows Attended Client activities to expose these settings at the Batch Process Step during configuration in Administration. The settings class must inherit from BatchView.ControlSettings. (See Settings)
Allow Flagged Folders If enabled, the batch can be completed with flagged folders.
Allow Unclassified Folders If true, the user will be allowed to complete the task while unclassified folders exist. If false, Classify Review will require that all folders at level immediately below the task be classified with a Content Type.
User Activity Frequency Set this property to a number of seconds. If a user has been inactive for this quantity of seconds, the current Processing Task will automatically close. This is intended to maintain user statistic integrity that could be damaged by users leaving the application open and unattended. Leaving this property set to zero will disable this feature. Value must be zero, or between 30 and 300.
UI Configuration
Command Configuration By default, all Object Commands will display on the context menu for each Batch View control. Editing this property allows administrators to hide Object Commands. Commands displayed in this listing will be those that apply to Batch Folder and Batch Page objects as well as those that apply to the Batch View controls used by the Activity.
