Grooper 2.70 Documentation

Image Review

Image Review is an attended activity which provides a user interface optimized for quickly reviewing and/or correcting the results of Classification or Separation. The user interface for the Image Review activity includes the Batch Viewer and Thumbnail Viewer Batch View controls.

Properties

Property Name Description
General
Pre-Defined Flag Messages A list containing pre-defined page flag messages. (See Lexicon)
Allowed IP Profiles IP Profiles available to operators during image review.
Require Review of All Pages When set to True, Batches or documents can not be completed unless all pages are marked as reviewed.
Allow Completion with Flagged Pages When set to True, Batches or documents with flagged pages can be completed.
User Activity Frequency Set this property to a number of seconds. If a user has been inactive for this quantity of seconds, the current Processing Task will automatically close. This is intended to maintain user statistic integrity that could be damaged by users leaving the application open and unattended. Leaving this property set to zero will disable this feature. Value must be zero, or between 30 and 300.
UI Configuration
Command Configuration By default, all Object Commands will display on the context menu for each Batch View control. Editing this property allows administrators to hide Object Commands. Commands displayed in this listing will be those that apply to Batch Folder and Batch Page objects as well as those that apply to the Batch View controls used by the Activity.

Commands

  Command Name Shortcut Keys Description
Apply Image Cleanup Alt+I Applies an IP Profile to one or more pages.
Property Name Description
IP Profile The IP Profile to execute.
Mark As Reviewed None Marks one or more pages as reviewed. If the page was previously flagged, the flag will be removed.
Unmark Ctrl+U Marks one or more pages as reviewed. If the page was previously flagged, the flag will be removed.
