Grooper 2.70 Documentation

Review

Provides a configurable user interface for human review of documents or pages in a batch. The Attended Client provides the user interface for this Activity. This client will display one or more Batch View controls as indicated by the Activity's configuration.

Properties

Property Name Description
UI Configuration
Batch Views Defines the collection of Batch Views that will be available for this Activity.
Command Configuration By default, all Object Commands will display on the context menu for each Batch View control. Editing this property allows administrators to hide Object Commands. Commands displayed in this listing will be those that apply to Batch Folder and Batch Page objects as well as those that apply to the Batch View controls used by the Activity.
Completion Criteria
Allow Completion with Flagged Pages Allow the completion of items that contain flagged pages.
All Pages at Level Allow the completion of items only when all Batch Pages are at this level. Leave the value set to zero (default) if items can be completed with pages at any level.
Folder Criteria Completion criteria options for specific Batch Folder levels. Configuring criteria here allows administrators to restrict the user's ability to complete tasks.
General
User Activity Frequency Set this property to a number of seconds. If a user has been inactive for this quantity of seconds, the current Processing Task will automatically close. This is intended to maintain user statistic integrity that could be damaged by users leaving the application open and unattended. Leaving this property set to zero will disable this feature. Value must be zero, or between 30 and 300.
