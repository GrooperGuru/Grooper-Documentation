Object Command

Object Commands perform specialized tasks on one or more Grooper Objects.

  Normalize Normalizes character data in the lexicon to the character set of the configured language. This command can be used to normalize imported lexicon data where words are spelled using characters from another language. For example, consider the Russian word 'ничего', which contains two Cyrillic characters ('е' and 'о') which are homoglyphs of the Latin characters 'e' and 'o'. While these characters look exactly the same, they are not the same Unicode characters. A lexicon entry spelled with Latin characters will not match a word extracted from a Russian document which is spelled with Cyrillic characters.

This command will scan all entries in the lexicon. If an extry contains characters which are invalid for the language, then a conversion will be attempted where each invalid character is converted to a corresponding homoglyph which is valid. If this conversion fails, the the entry is considered invalid.

If the conversion is successful, a lookup is performed to locate the controlling entry in the lexicon (i.e. the properly-spelled version). If the lookup fails, then the entry is considered an invalid entry. Otherwise, the controlling entry is updated with any frequency data from the invalid entry, and the invalid entry is discarded.

Batch Operations
  Combine Loose Pages Combines each range of contiguous loose pages into a new document, and attempts to classify each.
  Reprocess Batch Reprocess all or part of the batch through Classification.
  Revert To Loose Pages Moves all descendant page objects to the root of the selected folder, and then deletes all descendant folders.
Contents
  Add Multiple Items Creates multiple items as children of the selected object.
  Clear Children Deletes all children of the selected object(s).
Image
  Adjust Image Displays a UI allowing the user to edit the display properties of the image. For black and white images, the threshold may be adjusted. For grayscale and color images, the brightness and contrast may be adjusted.
  Apply Image Cleanup Applies an IP Profile to one or more pages.
  Crop Crops the displayed page to the current selection.
  Deskew Deskew Image. This command is provided for the shortcut key.
  Rotate Left Rotate Page Left. This command is provided for the statistics collection and the shortcut key.
  Rotate Right Rotate Page Right. This command is provided for the statistics collection and the shortcut key.
  Toggle Display Format Toggle the display format of the selected page between Color, Grayscale, and Binary.
  Undo Image Cleanup Replace the selected page(s) with its Undo Image.
Scan
  Append Starts the scanning process, appending newly-scanned pages to the end of the currently selected folder.
  Insert Starts the scanning process, inserting new pages immediately before or after the current selection.
  Replace Replace the currently selected pages with the result of a scan operation.
  Scan Starts the scanning process, appending newly-scanned pages to the end of the batch.
Send To
  Compressed Archive Exports the selected folders or documents to a ZIP archive.
  File System Exports the selected folders or documents to a file system directory.
  Mail Recipient Launches the default mail client, creating an email with the selected folders or documents attached.
  Printer Prints the selected items to a printer device.
  Test Batch Appends a copy of the selected items to a test batch.
Share
  Export to Compressed Archive Exports a set of Grooper nodes to a ZIP archive.
  Publish to Grooper Repository Publishes a set of Grooper Nodes to a Target Grooper Repository.
