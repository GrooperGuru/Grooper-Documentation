Grooper 2.70 Documentation

Object Command List

Object Commands perform specialized tasks on one or more Grooper Objects. The following list displays all Object Commands currently defined in Grooper:
Display Name Applies To Description
Add Multiple Items Grooper Node Creates multiple items as children of the selected object.
Add Selection to New Folder Batch Page, Batch Viewer Creates a new folder and moves all the selected Batch Pages inside it.
Adjust Image Batch Page, Batch Viewer Displays a UI allowing the user to edit the display properties of the image. For black and white images, the threshold may be adjusted. For grayscale and color images, the brightness and contrast may be adjusted.
Append Batch Folder, Scan Viewer Starts the scanning process, appending newly-scanned pages to the end of the currently selected folder.
Append to Previous Batch Object, Batch Viewer Append the selected object(s) to the previous folder. If the selected object is a folder, the contents of the selected folder will be appended to the previous folder, and the selected folder will be removed. If the selected object is one or more pages, t...
Apply Image Cleanup Batch Object, Image Review, IP Profile Applies an IP Profile to one or more pages.
Assign Content Type Batch Folder, Classification Viewer Assigns a Content Type to the currently selected folder.
Assign Document Type Batch Folder Assigns a Document Type to the currently selected folder.
Change File Attributes File System Link Changes the file attributes on the linked file.
Clear Children Grooper Node Deletes all children of the selected object(s).
Clear Flag Batch Object Clears any existing flags on selected items.
Clone To Production Batch, Batch Process Step Clones the batch into production, and launches it into a Batch Process.
Clone To Test Batch Clones this production batch to the 'Production Clones' folder of the test branch.
Combine Loose Pages Batch Folder, Classification Viewer Combines each range of contiguous loose pages into a new document, and attempts to classify each.
Compressed Archive Batch Object, Folder Content Generator Exports the selected folders or documents to a ZIP archive.
Copy File File System Link Writes a copy of the native file version to the file system. If a file with the same name already exists in the target folder, it will be overwritten.
Create Data Model Content Type Creates a new data model object on this content type.
Create Database Data Connection Creates a new database on the database server.
Create Local Resources Folder Content Type Creates a new data model object on this content type.
Crop Batch Page, Batch Viewer Crops the displayed page to the current selection.
Delete CMIS Link Deletes the CMIS document associated with the link and removes the link.
Delete File File System Link Deletes the linked file from the file system. Note: This will also delete the link from the batchfolder, since it is no longer valid.
Delete File FTP Link Deletes the linked file from the Ftp server. Note: This will also delete the link from the batchfolder, since it is no longer valid.
Delete File SFTP Link Deletes the linked file from the SSH/SFTP file server. Note: This will also delete the link from the batchfolder, since it is no longer valid.
Delete Message Mail Link Delete the linked Mail server object from the mail server. Note: This will also delete the link from the batchfolder, since it is no longer valid.
Deskew Batch Page, Batch Viewer Deskew Image. This command is provided for the shortcut key.
Download System Files Folder Downloads and installs the (system) files associated with this folder.
Expand Attachments Mail Link Expands all attachments as children of the folder object.
Expand Attachments Mail Message Expands all attachments as children of the folder object.
Export to Compressed Archive Grooper Node Exports a set of Grooper nodes to a ZIP archive.
File System Batch Object, Folder Content Generator Exports the selected folders or documents to a file system directory.
Flag Item Batch Object Marks the selected items as flagged.
Generate Control Sheets Content Type, Document Type Creates a new data model object on this content type.
Generate Table Content Type, Data Connection Generates a new database table from a Grooper Content Type.
Go To Next Invalid Document Batch Folder, Index Navigator Load the next invalid document.
Go To Next Issue Batch Object, Classification Viewer Sets the selection to the next loose page or un-classified Batch Folder.
Goto Next Flagged Item Batch Object, Batch Viewer Moves the focused item to the first flagged page of the batch (if any).
Import from Compressed Archive Grooper Root Imports objects from a Grooper ZIP archive file to the currently open Repository.
Insert Batch Object, Scan Viewer Starts the scanning process, inserting new pages immediately before or after the current selection.
Intersect Lexicon Creates a new lexicon containing all entries which appear both in this lexicon and a reference lexicon.
Load Content File System Link Loads the linked file and stores it locally in Grooper.
Load Content FTP Link Loads the linked file and stores it as the native document version.
Load Content Mail Link Loads the content from the linked Mail server object to the Grooper object.
Load Content SFTP Link Loads the linked file and stores it as the native document version.
Load Content CMIS Link Loads the content of the linked CMIS document and saves it to the Grooper document.
Load Properties CMIS Link Reads properties from the CMIS document and applies them to the Grooper document. Uses the Import Mappings defined on the CMIS Content Type.
Mail Recipient Batch Object, Folder Content Generator Launches the default mail client, creating an email with the selected folders or documents attached.
Mark As Reviewed Batch Page, Image Review Marks one or more pages as reviewed. If the page was previously flagged, the flag will be removed.
Merge Pages Batch Page, Classification Viewer Inserts the selected pages into a new folder, optionally classified as a content type.
Move File File System Link Writes the native file version to the file system.
Move Next Batch Folder, Index Navigator Go to the next document.
Move Objects Here File Store, Grooper Node Tests the connection to the file store.
Move Previous Batch Folder, Index Navigator Go to the previous document.
New Folder Batch Folder, Batch Viewer Adds a new Batch Folder.
Normalize Lexicon Normalizes character data in the lexicon to the character set of the configured language. This command can be used to normalize imported lexicon data where words are spelled using characters from another language. For example, consider the Russian word ...
Open MIME Type Handler Opens the document in the native application. If changes are saved to the document while it is open, the document in Grooper will updated to reflect the changes. If the document is printed to the Grooper Render Printer while it is open, the PDf version ...
Pause Batch Set the batch to a paused state.
Printer Batch Object Prints the selected items to a printer device.
Publish to Grooper Repository Grooper Node, Repository Connection Publishes a set of Grooper Nodes to a Target Grooper Repository.
Published Nodes Manager Grooper Root Displays the Published Nodes Manager, which allows previously published nodes to be unpublished or republished.
Purge Training Content Type Purges all classification training and samples from this item and all items below it.
Purge Training Data Element Purges all classification training and samples from this item and all items below it.
Rebuild Training Content Type Rebuilds classification training for this item and all items below it.
Remove Link Content Link Deletes the link between this object and the external content it is currently linked to.
Remove Native Version Batch Folder Deletes the locally-stored native file version from this folder object.
Remove PDF Version Batch Object Deletes the Grooper-generated PDF version from this item.
Replace Batch Page, Scan Viewer Replace the currently selected pages with the result of a scan operation.
Reprocess Batch Folder, Index Navigator Reprocess documents through extraction.
Reprocess Batch Batch Folder, Classification Viewer Reprocess all or part of the batch through Classification.
Reset Grooper Object, Property Grid Reset the selected property back to its default value.
Resume Batch Resumes processing on the batch at the current step.
Revert To Loose Pages Batch Folder, Classification Viewer Moves all descendant page objects to the root of the selected folder, and then deletes all descendant folders.
Rotate Left Batch Page Rotate Page Left. This command is provided for the statistics collection and the shortcut key.
Rotate Right Batch Page Rotate Page Right. This command is provided for the statistics collection and the shortcut key.
Save As... MIME Type Handler Saves the native file version to a file system location.
Save Content File System Link Overwrites the file system content with the content stored in Grooper.
Save Content FTP Link Writes the native file version to the Ftp file server
Save Content SFTP Link Writes the native file version to the SSH/SFTP file server
Save Content CMIS Link Updates the content of the linked CMIS document with the content of the Grooper document.
Save Properties CMIS Link Writes the Grooper document properties to the CMIS document. Uses the export mappings defined on the CMIS Content Type.
Save Version CMIS Link Adds a new version of the linked CMIS document from the content of the Grooper document.
Scan Batch Object, Scan Viewer Starts the scanning process, appending newly-scanned pages to the end of the batch.
Split Document Batch Page, Classification Viewer Splits a folder or document at the currently selected page, optionally assigning a content type to the new document.
Split Folder Batch Page, Batch Viewer Splits the content of the parent folder at the currently selected node, moving all items from the selected node to the end of the folder into a new folder.
Subtract Lexicon Removes all entries from this lexicon which appear in a reference lexicon.
Test Batch Batch, Batch Object Appends a copy of the selected items to a test batch.
Test Connection File Store Tests the connection to the file store.
Test Connection Data Connection Tests the data connection by making a connection to the server.
Toggle Display Format Batch Page Toggle the display format of the selected page between Color, Grayscale, and Binary.
Train Document Batch Folder, Classification Viewer Assign a Document Type for the selected document and train the Content Model with its features.
Truncate Lexicon Truncates the lexicon to the top N entries.
Unclassify Batch Folder, Classification Viewer Removes the selected folder and inserts its contents into the parent at the same position the folder was located.
Undo Image Cleanup Batch Page Replace the selected page(s) with its Undo Image.
Unmark Batch Page, Image Review Marks one or more pages as reviewed. If the page was previously flagged, the flag will be removed.
Update Columns Database Table Updates the columns associated with this Database Table. Use this command to update a table definition after changes have been made on the database server.
Update Process Batch, Batch Process Step Updates the Batch Process associated with this batch to the latest published version, and assigns a starting Batch Process Step.
View XML Batch Opens the selected batch in an XML view. Useful for comparing a batch to its control before and after changes are made.
