Row Match

Uses an extractor to match each row.

Inherits from: Table Extract Method

Properties

Property Name Description
General
Row Extractor
Type: Embedded Extractor
The extractor to be used to identify table rows and column values. One table row will be created for each match produced by this extractor. Column values can be populated using one of the following methods:
  • The Row Extractor defines regular expression groups which match the table column names. As regular expression group names do not support spaces, any spaces in the column name should be replaced with underscores (_) in the regular expression group name. For example, to capture a column named "Unit Price", use a regular expression group named as follows: (?<Unit_Price>...expression...)
  • The Row Extractor is a Data Type with child extractors having the same names as the the table columns.
  • If the Row Extractor does not produce a sub instance which maps to the column using one of the methods above, then the extractor defined on the Data Field object will be executed against the table row.
(See Embedded Extractor)
