Grooper 2.70 Documentation

FTP Link
Defines a reference to a file on an Ftp file server.

Inherits from: Content Link

Properties

Property Name Description
General
Relative File Path
Type: String
The path to the file relative to the ServerUri.
Error Message
Type: String
If set, indicates an error which occurred while attempting to access the remote object.
Server Connection
Server Uri
Type: String
The Url of a Base Ftp directory containing one or more other Ftp directories and/or files.
User Name
Type: String
The user name to be used for accessing the server.
Enable SSL
Type: Boolean, Default: False
 
Linked File Info
Last Write Time
The last write time of the linked file.
Length
The size of the linked file, in bytes.
Content Link Standard Values
Full Path
Type: String
The full path to the linked object.
Object Name
Type: String
The name of the linked object.
Path
Type: String
The path to the linked object, minus the name.
