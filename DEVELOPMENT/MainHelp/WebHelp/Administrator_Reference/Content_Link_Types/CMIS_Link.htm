Grooper 2.70 Documentation

CMIS Link
Defines a reference to a document in a CMIS Repository.

Inherits from: Content Link

Properties

Property Name Description
General
CMIS Repository
Type: CMIS Repository
This CMIS Repository in which the linked document resides. (See CMIS Repository)
CMIS Object Id
Type: String
The unique ID of the linked CMIS document.
Error Message
Type: String
If set, indicates an error which occurred while attempting to access the remote object.
Linked Document
Object Type Name
Type: String
The object's type name.
Object Type ID
Type: String
The object's type ID.
Base Type Name
Type: String
The object's base type.
Base Type ID
Type: String
The ID of the object's base type.
Content Stream File Name
Type: String
The filename associated with the document's content stream.
Content Stream MIME Type
Type: String
The MIME type associated with the document's content stream.
Content Stream Length
The length, in bytes, of the document's content stream.
Created By
Type: String
The name of the user who created the document.
Last Modified By
Type: String
The name of the user who last modified the document.
Created Date
The date the document was created.
Last Modified Date
The date the document was last modified.
Version Label
Type: String
The version label applied to the document.
Repository Name
Type: String
The name of the CMIS Repository in which the document exists.
Content Link Standard Values
Full Path
Type: String
The full path to the linked object.
Object Name
Type: String
The name of the linked object.
Path
Type: String
The path to the linked object, minus the name.
