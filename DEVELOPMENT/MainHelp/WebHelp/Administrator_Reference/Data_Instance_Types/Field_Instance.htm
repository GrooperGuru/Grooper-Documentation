Field Instance
Is Valid
Type: Boolean, Default: False
Indicates whether all descendant fields and/or table cells are valid.
Read Only
Type: Boolean, Default: False
Indicates whether the field value is editable in the user interface. If a Field Profile exists for this element, and the 'Read Only' property of that profile has a value, then this value is returned. Otherwise, the 'Read Only' property of the Data Field object is returned.
Required
Type: Boolean, Default: False
Indicates whether a value is required. If a Field Profile exists for this element, and the 'Required' property of that profile has a value, then this value is returned. Otherwise, the 'Required' property of the Data Field object is returned.
Visible
Type: Boolean, Default: False
Indicates whether a the user interface element associated with this data element is visible. If a Field Profile exists for this element, and the 'Visible' property of that profile has a value, then this value is returned. Otherwise, the 'Visible' property of the Data Field object is returned.
Base Type
The base type of this data instance.
Is Table Cell
Type: Boolean, Default: False
Returns True if this item is a cell in a table, or False if not.
Validate Message
Type: String
Indicates the reason why the element is not valid. When the IsValid property is false, this value will contain an error message indicating the reason.
Source Information
Index
Type: Int32, Default: 0
The starting index of this instance within the parent instance.
Length
Type: Int32, Default: 0
The length of this instance, in characters, within the parent instance. Please note that length represents the length of the element within the parent instance. It does not necessarily indicate the number of characters in the Value property or the number of items in the OcrResults object.
Page No
Type: Int32, Default: 0
The 1-based page number on which this item appears.
