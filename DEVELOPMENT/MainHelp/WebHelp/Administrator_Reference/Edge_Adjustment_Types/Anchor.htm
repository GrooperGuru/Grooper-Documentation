Grooper 2.70 Documentation

Anchor

Adjusts the edge relative to a text element on the document.

Inherits from: Edge Adjustment

Properties

Property Name Description
General
Anchor Extractor
Type: Embedded Extractor
Defines an extractor which matches the text to be used as an anchor. (See Embedded Extractor)
Scope
Type: ExecutionScope, Default: Global
Specifies what portions of the document will be searched for anchors. Can be one of the following values:
  • Global: The Anchor Extractor will execute against the entire document.
  • Parent: The Anchor Extractor will execute against the content of the parent element.
  • Internal: The Anchor Extractor will execute against the content of each instance returned by the Main Extractor.
Method
Type: AdjustmentMethod, Default: Inclusive
Specifies how the zone will be adjusted relative to the anchor. Can be one of the following values:
  • Inclusive: The zone will be adjusted so that it includes the entire region of the extracted result.
  • Exclusive: The zone will be adjusted so that it borders the extracted result, but does not include it.
Required
Type: Boolean, Default: False
Specifies what will happen if the extractor does not find an anchor. If this value is set to true, the section candidate will be considered invalid. Otherwise, the section will be extracted using pre-adjustment boundaries.
Manual Adjustment
Type: String
Defines a manual adjustment to be applied to the edge, after all other adjustments are made. Values can be positive or negative.
