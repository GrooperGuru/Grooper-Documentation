Grooper 2.70 Documentation

Fixed

Identifies section instances using a fixed geometric region. Outputs a single section instance containing all character data bounded by a rectangular region on a specific page. If the document does not contain the page specified, an empty result set will be returned.

Inherits from: Section Extract Method

Properties

Property Name Description
General
Bounds
Type: Rectangle, Default: (0,0):(0,0)
Defines the zone to be used for document types with no zonal profile for this section.
Page Number
Type: Int32, Default: 0
Indicates the 1-based page number from which the region should be extracted. If a negative is provided, this indicates a reference in reverse page order. For example, -1 indicates the last page, -2 indicates the second-to-last page, and so on.
