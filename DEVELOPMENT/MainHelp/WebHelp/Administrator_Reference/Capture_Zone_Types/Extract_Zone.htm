Grooper 2.70 Documentation

Extract Zone

Field Zones associate a Data Field with a region of the document, and optionally specify zonal extraction parameters. Field Zones can use OCR or barcode recognition to read a field value from the zone.

They are created as the child of an Extraction Profile, and can have Registration Zones as children.

Properties

Property Name Description
Extraction
Extraction Type The type of data extraction to be used with this zone. Can be one of the following values:
  • None: No zonal extraction will be performed. This zone simply establishes the zoom area when the user tabs into the field.
  • Barcode: The zone will use barcode recognition to read a barcode value.
  • OCR: The zone will use OCR to read a text value.
  • FullText: The zone will be re-extracted from existing full text OCR, if it exists.
OCR Profile The OCR Profile to be used for zonal data extraction. (See OCR Profile)
Barcode Detection Settings The settings to be used for barcode recognition. (See Barcode Detection)
Apply Field Extractor Controls whether the extractor for the field will be executed on the OCR results obtained from the zone.
Shape Name The name of a Shape to be used as the region for this zone.
Zone Location
Page No The page number associated with this zone. A negative value indicates on offset starting at the end of the document. For example, -1 indicates the last page of the document -2 indicates the 2nd to last page of the document, and etc.
Bounds The location of the zone.
Image Cleanup
IP Profile The IP Profile to run prior to zonal extraction. This IP Profile performs temporary image cleanup prior to extraction. It does not permanently alter the image. (See IP Profile)
