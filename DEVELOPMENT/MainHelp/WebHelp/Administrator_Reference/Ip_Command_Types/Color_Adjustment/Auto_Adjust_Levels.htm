Auto Adjust Levels
The Adjust Levels command corrects the tonal range and color balance by adjusting the shadow, midtone, and highlights of an image.

Inherits from: IP Command

Properties

Property Name Description
General
Gamma R
Type: Double, Default: 1
Gamma Adjustment - Red
Gamma G
Type: Double, Default: 1
Gamma Adjustment - Green
Gamma B
Type: Double, Default: 1
Gamma Adjustment - Blue
Minimum Color
Type: Color, Default: 0, 0, 0
Gets or sets the black point in the image used as an input clipping point.
Mid Color
Type: Color, Default: 128, 128, 128
Gets or sets the gray point in the image used as a gamma adjustment.
Maximum Color
Type: Color, Default: 255, 255, 255
Gets or sets the white point in the image used as an input clipping point.
Minimum Output
Type: Color, Default: 0, 0, 0
Gets or sets the dark color value to stretch the histogram to.
Maximum Output
Type: Color, Default: 255, 255, 255
Gets or sets the light color value to stretch the histogram to.
