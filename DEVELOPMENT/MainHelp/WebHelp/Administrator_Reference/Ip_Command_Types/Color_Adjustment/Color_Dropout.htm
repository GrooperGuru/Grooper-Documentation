Color Dropout
Replaces pixels in a specific color range with a replacement color. This command has several possible uses:
  • Replace all colors. - Leave the Target Color empty.
  • Replace all shades of gray. - Leave the Target Color empty, and set Invert to True.
  • Replace a specific color - Set the Target Color to the color which should be replaced.
  • Preserve a specific color - Set the Target Color to the color which should be preserved, and set Invert to True.


Inherits from: IP Command

Properties

Property Name Description
General
Target Color
The color to match in the image. If this value is empty, the algorithm will match all shades of gray from black to white.
Fill Color
The color to be used to replace dropped out colors. If no color is specified, the dominant background color of the document will be used.
Maximum Variation
Type: Double, Default: 20%, Range: 0% - 100%
Specifies a maximum deviation from the Dropout Color, or a maximum deviation from pure gray in the case where no Dropout Color is specified.
Invert
Type: Boolean, Default: False
If true, the logic for deciding whether to drop pixels out will be inverted.
Command Info
Command Type
Type: String
The type of IP command.
Supported Pixel Formats
Type: String
The pixel formats supported by this command.
