Grooper 2.70 Documentation

Rotate
Rotates an image by a given angle.

Inherits from: IP Command

Properties

Property Name Description
General
Angle
Type: Double, Default: 0°, Range: 0° - 360°
The angle of the rotation, in degrees.
Interpolation Mode
Type: RotateInterpolationMode, Default: Linear
The interpolation mode controls the output quality and speed of the rotate operation. Can be one of the following values:
  • Linear: Linear interpolation is slower but more accurate than the Nearest Neighbor interpolation. It is faster but less accurate than Cubic Interpolation. Linear interpolation sets the destination pixel based on analysis of the surrounding 4 pixels in the source image.
  • Cubic: The most accurate, but slowest interpolation method. Sets destination pixels based on a 16-pixel neighborhood in the source image.
  • NearestNeighbor: This is the fastest and least accurate interpolation mode. This simply sets the destination pixel to the value of the closest source pixel.
  • None - Fastest, lowest quality.
  • Bilinear - Fast, better quality.
  • Bicubic - Slower, better quality.
Resize
Type: Boolean, Default: True
If set to True, the command resizes the image to fit with in the dimensions of the image. If the command is set to False, the image may be cropped to fit within the dimensions of the image.
Background Color
The color used to fill in edges exposed by rotation. If no background color is specified, the dominant background color of the image will be used.
Command Info
Command Type
Type: String
The type of IP command.
Supported Pixel Formats
Type: String
The pixel formats supported by this command.
