Grooper 2.70 Documentation

Resize
Resizes an image.

Inherits from: IP Command

Properties

Property Name Description
General
Sizing Method
Type: SizeMethod, Default: ScaleFactor
Specifies the type of resize operation to be applied. Can be one of the following values:
  • ScaleFactor: Based on a percentage of the original image size.
  • TargetResolution: Based on a target resolution for the image, in DPI.
  • ChangeResolutionOnly: Does not change the size of the image - simply changes the resolution to the target resolution. This can be used to correct the resolution on images to a fixed value.
  • CustomSize: Resizes the image to an arbitrary size.
Scale Factor
Type: Double, Default: 100%, Range: 0% - 100%
The factor by which to scale the image when Sizing Method is ScaleFactor.
Target Resolution
Type: Double, Default: 200, Range: 25 - 900
Target resolution for the image when Sizing Method is TargetResolution or ChangeResolutionOnly. Units are in DPI (dots per inch).
Custom Size
Type: Unit Size
The size to which the image should be resized when the Sizing Method is set to CustomSize.
Interpolation Mode
Type: ResizeInterpolationMode, Default: Linear
The interpolation mode controls the output quality and speed of the resize operation. Can be one of the following values:
  • Linear: Linear interpolation is slower but more accurate than the Nearest Neighbor interpolation. It is faster but less accurate than Cubic Interpolation. Linear interpolation sets the destination pixel based on analysis of the surrounding 4 pixels in the source image.
  • Cubic: The most accurate, but slowest interpolation method. Sets destination pixels based on a 16-pixel neighborhood in the source image.
  • NearestNeighbor: This is the fastest and least accurate interpolation mode. This simply sets the destination pixel to the value of the closest source pixel.
  • SuperSampling: Sets destination pixels based on a weighted sum of the source image. This interpolation mode provides the best results in cases where the destination image is significantly smaller than the source image.
  • Lanczos: Lanczos interpolation.
