Grooper 2.70 Documentation

Photo Removal
Automatically detects and removes photographs.

Inherits from: IP Command

Properties

Property Name Description
General
Mask Size
Type: String, Default: 4pt
Controls the amount of entropy smoothing applied during photo detection. This value is used to produce the Entropy image from the Delta image. The Delta image represents the amount of change in the image at each pixel location. In the smoothed Entropy image, each pixel value is a measurement of entropy in the local pixel neighborhood of MaskSize x MaskSize pixels.
Minimum Entropy
Type: Double, Default: 25%, Range: 0% - 100%
Adjusts the entropy sensitivity. Lower values will detect lower levels of entropy. This value represents an intensity threshold which is applied to the Entropy image to produce the Candidates image. This value should be tuned to produce a distinct rectangular blob on the Candidates image for each photo.

Type: Region Settings, Default: 1in, 0.25inÂ², 50%
Defines setting which control how photo regions are detected.
Detect Line Art Photos
Type: Boolean, Default: True
If enabled, line art photos will also be detected.
