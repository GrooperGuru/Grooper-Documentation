Grooper 2.70 Documentation

Dilate Erode
Applies a dilation (thicken) or erosion (thinning) filter to the image. A dilation operation dilates the feature pixels in an image, as shown below.

An erosion operation erodes the feature pixels in an image, as shown below.



Inherits from: IP Command

Properties

Property Name Description
General
Operation
Type: OperationType, Default: Dilate
Specifies whether the image should be dilated or eroded. Can be one of the following values:
  • Dilate: Dilates the feature pixels in the image.
  • Erode: Erodes the feature pixels in the image.
Factor
Type: Int32, Default: 1, Range: 1 - 16
The factor by which dilation or erosion is performed.
Command Info
Command Type
Type: String
The type of IP command.
Supported Pixel Formats
Type: String
The pixel formats supported by this command.
