Grooper 2.70 Documentation

Filter
Applies a filter to an image.

Inherits from: IP Command

Properties

Property Name Description
General
Filter Type
Type: FilterTypeEnum, Default: Average
Sets the filter type that is applied to the image. Can be one of the following values:
  • Average
  • Median
  • MedianCross
  • Max
  • Min
  • Laplacian
  • Gaussian
  • HighPass
  • LowPass
  • Sharpen
Mask Width
Type: String, Default: 5px
The width of the mask.
Mask Height
Type: String, Default: 5px
The height of the mask.
Mask Size
Type: String, Default: 5px
The size of the mask.
Predefined Mask Size
Type: FilterMaskSize, Default: ippMskSize3x3
Specifies a predefined mask size for the mask used for filtering. Can be one of the following values:
  • ippMskSize3x3
  • ippMskSize5x5
Sigma
Type: Single, Default: 1.6
The size of the mask.
Command Info
Command Type
Type: String
The type of IP command.
Supported Pixel Formats
Type: String
The pixel formats supported by this command.
