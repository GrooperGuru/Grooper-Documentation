Grooper 2.70 Documentation

Classify Image
Compares an image against a set of sample images and classifies the image based upon which sample image it is found to be most similar to.

Inherits from: IP Command

Properties

Property Name Description
General
Sample Images
Type: Sample Image Collection
One or more sample images representing the shape to be detected. This property cannot be edited while changes are pending. (See Sample Image Collection)
Minimum Similarity
Type: Double, Default: 85%, Range: 0% - 100%
The minimum percentage of similarity required for successful image classification.
Color Space
Type: ColorSpaceType, Default: RGB
The color space to use. Can be one of the following values:
  • RGB: RGB Color Space - Channels: Red, Green, Blue.
  • HSV: HSV Color Space - Channels: Hue, Saturation, Value (Intensity).
  • LAB: LAB Color Space - Channels: Luminosity, A Value, B Value.
  • YCbCr: YCbCr Color Space - Channels: Luma, Chroma Blue, Chroma Red.
  • YUV: YUV Color Space - Channels: Y (luma), U (chroma red), V (chroma blue).
  • XYZ: XYZ Color Space - Channels: X, Y, Z.
  • HLS: HLS Color Space - Channels: Hue, Lightness, Saturation.
The input image will be converted to the indicated color space prior to channel extraction.
Default Image Class
Type: String, Default: None
The value to be used for the ImageClass output value when the image matches none of the predefined Image Classes.

The type of IP command.

The pixel formats supported by this command.
