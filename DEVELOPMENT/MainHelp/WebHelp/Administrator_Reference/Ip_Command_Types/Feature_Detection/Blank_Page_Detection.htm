Grooper 2.70 Documentation

Blank Page Detection
Determines if a page is blank based on a threshold of black pixels.

Inherits from: IP Command

Properties

Property Name Description
General
Black Pixel Threshold
Type: Double, Default: 2%, Range: 0% - 100%
The percentage of black pixels below which the image is considered blank.
Ignore Border Size
Type: Double, Default: 0.25, Range: 0 - 8
The size, in inches, of the border area which will be ignored for the purposes of determining whether the page is blank.
Processing Resolution
Type: ProcessingResolution, Default: Dpi75
The image resolution at which this command will be processed. The image will be resampled to the specified resolution before processing. Downsampling to a lower resolution can increase the speed of the command, sometimes at the expense of accuracy. Can be one of the following values:
  • FullResolution
  • Dpi10
  • Dpi15
  • Dpi20
  • Dpi25
  • Dpi30
  • Dpi40
  • Dpi50
  • Dpi75
  • Dpi100
  • Dpi150
  • Dpi200
  • Dpi250
  • Dpi300
Command Info
Command Type
Type: String
The type of IP command.
Supported Pixel Formats
Type: String
The pixel formats supported by this command.

Result Type

Member Name Member Type Description
IsBlank Boolean Indicates whether the image was determined to be blank.
BlackPixelPercentage Double The percentage of pixels in the image considered to be black pixels.
